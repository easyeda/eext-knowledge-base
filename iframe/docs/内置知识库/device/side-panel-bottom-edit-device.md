# 编辑器件

对创建完成的器件进行编辑修改。

**编辑器件步骤**：

-   打开底部面板或放置器件弹窗，找到需要进行编辑的器件，<kbd>鼠标右键</kbd> - 编辑器件。

**说明**

-   系统库的器件是无法编辑的，需要编辑请另存后再编辑。

弹出器件的编辑弹窗，弹窗与新建器件弹窗一样，修改完成后点击确认即可更新信息。

![image-20211221170505268](/storage/images/cn/device/side-panel-bottom-edit-device/side-panel-bottom-edit-device_20240621_172123.png)

修改完成器件后，如果列表没有及时更新，可以需要点击器件库<kbd>刷新</kbd> 按钮后看到最新修改的信息。
