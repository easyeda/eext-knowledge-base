# 放置器件

**操作方法**：

-   顶部面板 - 放置 - 器件

-   快捷键<kbd>F</kbd>调出底部面板的元件库列表

-   快捷键<kbd>Shifi</kbd>+<kbd>F</kbd> 打开独立放置器件对话框

选择器件点击选择需要的器件点击右侧的放置按键，鼠标光标变成十字状态即可放置器件，放置器件不支持拖放。
