# IPC/DAC-2552属性

当默认符号库管理模式是简易模式时，我们可以给器件库增加IPC/DAC-2552属性。

入口：打开元件符号 - 顶部菜单 - 工具 - IPC/DAC-2552属性

![picture 0](/storage/images/cn/device/tools-ipc-dac-2552/tools-ipc-dac-2552_20240621_172123.jpg)

使用方法：
在打开对话框后，中间部分可以按需填写对应的属性，左下角可以导入已有的2552属性xml文件，也可以把当前的属性参数导出为xml文件

![](/storage/images/cn/device/tools-ipc-dac-2552/tools-ipc-dac-2552_20240625_140500.png)

当打开原理图的时候，也可以在工具菜单查看和编辑元件的2552属性

![picture 2](/storage/images/cn/device/tools-ipc-dac-2552/tools-ipc-dac-2552_20240621_172124.jpg)
