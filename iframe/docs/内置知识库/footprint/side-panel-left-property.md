# 封装属性

封装界面右侧面板是显示整个封装的属性、位号、分类已经其他的信息。

![image-20211222103941209](/storage/images/cn/footprint/side-panel-left-property/side-panel-left-property_20240621_172123.png)

**封装**：用来对封装的命名，把旁边的复选框打上勾后，可以在封装里面显示封装的名称。

![image-20211222104021402](/storage/images/cn/footprint/side-panel-left-property/side-panel-left-property_20240621_172124.png)

**位号**

对封装进行位号设置。

**分类**

在这里可以修改封装的分类，方便后期的管理和维护。

鼠标点击分类的输入框，弹出选择弹窗，即可在这里修改分类。

![image-20211222104241894](/storage/images/cn/footprint/side-panel-left-property/side-panel-left-property_20240621_172125.png)

**描述**

用于对封装增加一些文字描述。

![image-20211222104341108](/storage/images/cn/footprint/side-panel-left-property/side-panel-left-property_20240621_172126.png)

**属性添加**

用于对焊盘的属性添加，单击右侧的下拉选择框，点击需要添加的属性，即可完成添加。

![image-20211222104422627](/storage/images/cn/footprint/side-panel-left-property/side-panel-left-property_20240621_172127.png)

将鼠标悬停在添加属性的空白处中间，点击<kbd>鼠标右键</kbd>可删除添加的属性，系统默认属性不可删除。

![image-20211222104519165](/storage/images/cn/footprint/side-panel-left-property/side-panel-left-property_20240621_172128.png)

嘉立创EDA支持将封装与3D模型进行关联。

**功能入口**：

- 左侧面板 - 库设计 - 属性

![图 0](/storage/images/cn/footprint/side-panel-left-property/side-panel-left-property_20250331_173835.jpg)

在属性设置中，点击“3D模型”输入框，可打开3D模型管理器界面。通过该界面的模型库列表，用户能够选择或替换绑定的3D模型。

![图 1](/storage/images/cn/footprint/side-panel-left-property/side-panel-left-property_20250331_174110.jpg)

完成3D模型绑定后，所选模型的名称将显示在对应的输入框中。

![图 2](/storage/images/cn/footprint/side-panel-left-property/side-panel-left-property_20250331_174140.jpg)

对于已绑定模型的封装，用户可以在底部面板预览所绑定的3D模型，以确保其符合设计需求。

![图 3](/storage/images/cn/footprint/side-panel-left-property/side-panel-left-property_20250603_092912.jpg)

若器件属性中绑定了新的3D模型，则优先使用器件模型。用户可通过右键点击模型输入框，选择还原为封装模型。

![图 4](/storage/images/cn/footprint/side-panel-left-property/side-panel-left-property_20250603_093020.jpg)
