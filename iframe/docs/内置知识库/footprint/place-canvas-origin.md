# 画布原点

将绘制好的封装设置原点位置

**放置光标原点步骤**：

-   顶部菜单 - 放置 - 画布原点 - 从光标

## 从光标

选择后，可用鼠标来选择原点位置，选择后位置后，那里的坐标值将会变为零。

![image-20211222104643504](/storage/images/cn/footprint/place-canvas-origin/place-canvas-origin_20240621_172123.png)

## 从坐标点

点击后会弹出一个对话框，在里面输入想要设置为原点的坐标值，即可将你输入的坐标值调整为原点。

![输入坐标原点](/storage/images/cn/footprint/place-canvas-origin/place-canvas-origin_20240621_172124.gif)

## 从焊盘中央

点击后，系统就会选择两边焊盘的中间为原点。

![image-20211222112304303](/storage/images/cn/footprint/place-canvas-origin/place-canvas-origin_20240621_172125.png)

![image-20211222112337029](/storage/images/cn/footprint/place-canvas-origin/place-canvas-origin_20240621_172126.png)
