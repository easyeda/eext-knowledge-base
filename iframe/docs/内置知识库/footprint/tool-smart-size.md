# 智能尺寸

嘉立创EDA专业版的智能尺寸与标准版的智能尺寸功能相同，都是用于测量和快速绘制封装。

**使用方法**：

-   顶部菜单 - 工具 - 智能尺寸

![image-20211222142030372](/storage/images/cn/footprint/tool-smart-size/tool-smart-size_20240621_172123.png)

焊盘四周出现白色的小圆圈时，智能尺寸就已开启成功。

**说明**：

-   同一个焊盘的测量点是无法选择的
