# 检查尺寸

检查尺寸功能用于快速的检查封住的尺寸大小，与智能尺寸不同的是，检查尺寸是不能进行编辑，但显示的内容会比智能尺寸显示的信息多。

**使用方法**：

-   顶部菜单 - 工具 - 检查尺寸

![image-20211222164043720](/storage/images/cn/footprint/tool-check-dimension/tool-check-dimension_20240621_172123.png)

**说明**：

-   检查尺寸显示的内容是根据当前器件来生成的，器件修改或编辑后会自动退出检查功能。
