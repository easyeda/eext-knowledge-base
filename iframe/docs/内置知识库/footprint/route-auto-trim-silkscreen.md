# 自动裁剪丝印

自动裁剪丝印是把绘制在焊盘上的丝印裁剪掉。用于在快速绘制封装外形丝印时。

**操作方法**：

-   顶部菜单 - 布线 - 剪裁丝印

![image-20211222135510108](/storage/images/cn/footprint/route-auto-trim-silkscreen/route-auto-trim-silkscreen_20240621_172123.png)

**说明**：

-   自动裁剪丝印只能用于折线；矩形、圆型等图元无法生效使用
-   剪裁丝印功能默认启动
