# 器件关联

器件需要关联封装和符号才可以被放置在原理图或者PCB。

![图 0](/storage/images/cn/footprint/side-panel-left-device/side-panel-left-device_20241125_113633.png)

符号或者封装界面右侧面板的器件标签页是展示这个封装被哪些器件所关联，方便新建器件关联当前封装，也方便直接在已有绑定的器件中，直接移除这个封装的绑定关系。

![图 2](/storage/images/cn/footprint/side-panel-left-device/side-panel-left-device_20240621_172123.png)
