# 导入焊盘坐标文件

## 功能入口

-   顶部菜单- 文件 - 导入 - 焊盘坐标文件

![图 0](/storage/images/cn/footprint/file-import-pads-pick-and-place-file/file-import-pads-pick-and-place-file_20240621_172123.jpg)

![图 1](/storage/images/cn/footprint/file-import-pads-pick-and-place-file/file-import-pads-pick-and-place-file_20240621_172124.jpg)

然后导入表格文件
![图 3](/storage/images/cn/footprint/file-import-pads-pick-and-place-file/file-import-pads-pick-and-place-file_20240621_172125.jpg)
