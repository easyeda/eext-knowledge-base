# 多人实时协作

嘉立创EDA支持多人实时协作功能，允许多名成员在统一工作区内同步操作并即时共享操作结果，该功能仅面向企业私有部署版本提供。
**功能入口**：

当多名用户同时打开同一工程时，自动进入多人实时协作模式。

![图 0](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_105006.jpg)

界面右上角显示参与协同的用户头像。每个用户的头像旁会以不同颜色标识，表示该用户在操作时的选中状态。

![图 2](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_105053.jpg)

在进行多人实时协作之前，请确保所有参与操作的用户均拥有对该项目的编辑权限。

![图 1](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_105045.jpg)

## 工程协同

多人协作期间，工程文档树的任何更改都将实时同步给所有协作者。

![图 3](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_110604.gif)

生成或更新复用模块时，整个工程也会同步更新。

![图 4](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_111455.gif)

## 原理图协同

多人协作期间，支持同步图页图纸的属性变更。

![图 6](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_113144.gif)

用户对原理图图元的放置、移动、旋转、删除及属性修改也将同步给其他协作者。

![图 7](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_113711.gif)

## PCB 协同

多人协作期间，支持 PCB 图层的数据同步。

![图 8](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_135802.gif)

用户对 PCB 图元的放置、移动、旋转、删除及属性修改也将实时同步给其他协作者。

![图 9](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_141030.gif)

支持PCB的设计规则同步。

![图 10](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_141309.gif)

## 工程历史

多人协作期间，协作者的操作记录可以在顶部菜单栏“文件”下的“工程历史”中查看。

![图 11](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_141531.jpg)

点击“工程历史”选项后，右侧弹窗将显示工程的修改记录，其中包括具体的修改时间和修改者信息。请注意，修改记录并非实时同步，而是基于用户点击“工程历史”时的最新记录进行展示。

![图 12](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_141758.jpg)

支持还原工程和对比记录，方便团队成员追溯和管理版本变更。

![图 13](/storage/images/cn/project/real-time-multi-person-collaboration/real-time-multi-person-collaboration_20250304_141859.jpg)
