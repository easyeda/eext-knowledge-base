# 十四 包装

&emsp;&emsp;拿到QC传送过来的电路板，首先会打印一个装箱条码。在装箱条码上，会有一些重要的信息，有些客户在嘉立创下单的时候，选择了一些个性化的参数，比如，包装要求是选择了嘉立创标识盒子还是空白无字盒子。使用气泡布抽真空后放置一小袋干燥剂包装发货。如图14-1所示为气泡布抽真空示意图，真空包装机器设备如图14-2所示。

&emsp;&emsp;如果客户的板子数量比较多，会分成几叠，在每一叠上面贴上对应的标签，标签上有电路板的批次、每包数量、一共数量等信息。真空包装完毕，就该装箱了，这个是一个小批量的单。在装箱之前，在箱子的四个角，已经放置了泡沫固脚，防止电路板在运输过程中或者搬运过程中的磕磕碰碰，如图14-3所示
