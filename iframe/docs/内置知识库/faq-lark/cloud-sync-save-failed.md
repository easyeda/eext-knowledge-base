# 提示同步云端数据保存失败

## 问题
提示同步云端数据保存失败，工程已锁定，如果需要修改，保存，请联系工程所有者或是管理员。

## 解决方法
如果是半离线，可能是只读了，文件变为只读了如何操作图文教程您看下： https://prodocs.lceda.cn/cn/faq/client/#%E5%8F%AA%E8%AF%BB1. 再不行可以保留下当前工程：下载当前无法保存文件的源码（每个原理图页，PCB页单独下单源码），然后在本地找到这个eprj文件取消只读，并打开粘贴原来无法保存的源码进去就恢复到原来的文件内容并也可以保存了。(优先另存为本地） 2.息屏了，登录信息退出了重新登录一下 3.工程名有特殊符号 4.如果是封装，把封装名改一下再保存。
