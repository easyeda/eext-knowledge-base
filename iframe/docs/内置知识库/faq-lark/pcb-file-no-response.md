## 问题
双击打开PCB，无反应

## 解决方法
1、请确保电脑有显卡，并且不是很老的显卡。因为打开PCB需要显卡的支持。2、重新打开编辑器试试或退出登录（选择移除缓存），如果还是不能打开，可能有数据异常3、如果是客户端，请使用谷歌浏览器并开始硬件加速。目前发现有些win7电脑安装客户端无法正常打开PCB，浏览器则正常。打开PCB文件没有反应如何处理

## 问题
eprj 格式打开后只在软件首页是什么原因？

## 解决方法
1. 2.1后的版本做了检测，文件只读的状态下只能进入软件页面，无法进入编辑，选中文件右键点击属性，把只读取消勾选就可以进入编辑了   2. 可能用户直接把epro文件重名为eprj，导致无法打开
