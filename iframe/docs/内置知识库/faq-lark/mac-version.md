# MAC版本

## 问题
MAC系统安装后无法启动怎么解决？

## 解决方法
首先在命令行输入ls -al /User/xxxx/.config查看权限，例如用户名是11，那就是在命令行输入ls -al /User/11/.config就可以看到有没有权限了 解决方法1. 手动创建LECDA-PR0 目录 方法2. 将 /User/xxxx/./config 目录归属设置为当前用户 方法3. 提升立创EDA的运行权限到 root

## 问题
MAC版本的触摸板手势左右正常，上下变成放大缩小了

## 解决方法
滚轮的功能就相当于在触摸屏上双指上下拖动，改为 CTRL +滚轮 按住ctrl再上下拖动，就是客户想要的拖动板子
