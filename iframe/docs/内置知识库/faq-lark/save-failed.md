# 保存失败

## 问题
PCB页面保存失败

## 解决方法
把任务管理器里的所有客户端进程关闭，再重启，包括标准版的进程

---

## 问题
工程文件保存时日志中提示SQLITE IOERR: disk I/O error

## 解决方法
1：获取工程源码，复制到任意文本中 2：把本地工程文件暂时复制到另外一个文件夹 3：删除原有文件夹，再新建一个同名文件夹 4：将刚才的工程文件复制进来 5：打开文件后粘贴工程源码，再应用

---

## 问题
器件管理器里分配商城器件成功后，会概率性频繁的提示错误: SQLITE READONLY: attempt to write a readonly database，并保存失败，重启后文件会无法打开，文件没有只读怎么办？

## 解决方法
看下电脑是不是有同步软件之类的，如果有请关闭此类软件再看下
