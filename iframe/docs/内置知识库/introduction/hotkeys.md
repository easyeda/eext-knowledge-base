# 快捷键

嘉立创EDA专业版为用户提供了一系列可自定义的快捷键，旨在提高工作效率。用户可以根据个人需求对每个快捷键进行个性化设置。

快捷键配置界面入口：

顶部菜单 - 设置 - 快捷键

![图 108](/storage/images/cn/introduction/hotkeys/hotkeys_20240621_172123.png)

**注意**：

- 自v1.8版本起，系统新增了单键呼出顶部菜单的功能，以实现连续按键操作。因此，之前设置的部分单键功能可能会被清空，以便为顶部菜单预留单键。用户可以在确保不与其他快捷键冲突的前提下，在快捷键设置中重新配置单键功能。

## 修改快捷键

双击快捷键表格，将弹出快捷键配置窗口，用户可以在该窗口中设置全局快捷键和菜单快捷键。

![图 3](/storage/images/cn/introduction/hotkeys/hotkeys_20250221_095844.gif)

支持切换快捷键风格，目前有嘉立创EDA专业版、嘉立创EDA标准版、Altium Designer的快捷键风格。未来将陆续增加更多风格选项。

![图 1](/storage/images/cn/introduction/hotkeys/hotkeys_20250221_094930.jpg)

点击“恢复默认”按钮，可以将当前页面的所有快捷键设置恢复为默认，并自动切换至嘉立创EDA专业版的快捷键风格。

![图 2](/storage/images/cn/introduction/hotkeys/hotkeys_20250221_095035.jpg)
