# 导出DXF

嘉立创EDA支持导出原理图的 DXF 文件。

**功能入口**：

- 顶部菜单栏 - 导出 - DXF
- 顶部菜单栏 - 文件 - 导出 - DXF

![图 2](/storage/images/cn/schematic/export-dxf/export-dxf_20250321_105346.jpg)

![图 0](/storage/images/cn/schematic/export-dxf/export-dxf_20250321_105328.jpg)

# 使用方法

在导出 DXF 文件的设置窗口中，用户可选择导出范围，支持以下选项：全部原理图、单独原理图、当前图页。

![图 3](/storage/images/cn/schematic/export-dxf/export-dxf_20250321_110018.jpg)

支持在导出选项中选择是否包含图纸。

![图 4](/storage/images/cn/schematic/export-dxf/export-dxf_20250321_110244.jpg)
