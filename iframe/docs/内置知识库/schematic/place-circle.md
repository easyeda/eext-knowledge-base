# 圆形

可以在在原理图绘制一个圆,圆形的绘制方式为中心圆。

**操作入口：**

-   顶部菜单 - 放置 - 圆形

-   快捷键<kbd>Alt</kbd>> + <kbd>C</kbd>>

![图 0](/storage/images/cn/schematic/place-circle/place-circle_20240621_172124.png)

先确定圆的中心点，再往外拉伸，鼠标点击左键，即可完成圆形的绘制。

![image-20201219140740779](/storage/images/en/schematic/place-circle/place-circle_20240621_172124.png)

选择圆形可以在右侧的属性面板修改位置、尺寸、样式等设置。还可以一键恢复为主题的样式。

![图 1](/storage/images/cn/schematic/place-circle/place-circle_20240621_172125.png)
