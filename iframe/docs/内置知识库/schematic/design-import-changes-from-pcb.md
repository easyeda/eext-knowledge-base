# 从PCB导入变更

从PCB导入变更只能对已有元件进行新增、修改、删除属性、替换元件，不能修改网络。

需要增删的元件会在日志中体现，但需要用户手动放置或删除元件。

![图 2](/storage/images/cn/schematic/design-import-changes-from-pcb/design-import-changes-from-pcb_20240621_172124.png)

当PCB有修改希望更新回原理图时，在PCB可以点击 **顶部菜单 > 设计 > 更新PCB到原理图**
或在原理图点击 **顶部菜单 > 设计 > 从PCB导入变更**

![图 0](/storage/images/cn/schematic/design-import-changes-from-pcb/design-import-changes-from-pcb_20240621_172125.png)

![图 1](/storage/images/cn/schematic/design-import-changes-from-pcb/design-import-changes-from-pcb_20240621_172126.png)

**注意**：要从PCB更新至原理图，原理图和PCB都必须在同一个板子下。
