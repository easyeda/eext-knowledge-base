# 连接点标识

嘉立创EDA专业版提供“连接点标识”功能。此功能旨在简化交叉导线的绘制操作，提升设计效率。

### 功能入口

1. **顶部菜单栏**：选择“放置” > “连接点标识”（快捷键：`Alt` + `J`）。
   ![图 0](/storage/images/cn/schematic/place-junction-flag/place-junction-flag_20251201_113544.png)

# 操作步骤

1. 通过顶部菜单栏选择“放置” > “连接点标识”。
2. 将连接点标识放置于导线交叉处，即可完成连接。
   ![图 1](/storage/images/cn/schematic/place-junction-flag/place-junction-flag_20251201_114505.gif)
