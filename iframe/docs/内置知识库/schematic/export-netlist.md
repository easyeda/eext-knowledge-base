# 导出网表

嘉立创EDA专业版目前只支持导出部分EDA工具的网表文件格式。可导出单一文档网表也可导出整个工程的网表。

-   **顶部菜单 - 文件 - 导出 - 导出网表**

![图 156](/storage/images/cn/schematic/export-netlist/export-netlist_20240621_172124.png)

或者：**顶部菜单 - 导出 - 网表**

![图 157](/storage/images/cn/schematic/export-netlist/export-netlist_20240621_172125.png)

弹出窗口可以修改导出网表类型,需要导出网表的页面范围以及设置器件和封装属性。

![图 1](/storage/images/cn/schematic/export-netlist/export-netlist_20240929_155852.png)

点击器件和封装属性右边窗口后，可以选择需要导出的信息。
![图 3](/storage/images/cn/schematic/export-netlist/export-netlist_20240929_160314.png)

目前支持导出嘉立创EDA，Allegro，PADS 9.5 版本的网表。
