# 文件源码

嘉立创EDA支持导出或编辑文件的源码。

-   顶部菜单 - 文件源码

![image-20210522174014285](/storage/images/en/schematic/file-file-source/file-file-source_20240621_172123.png)

点击下载后即可下载工程的文件源码，然后可以通过文本编辑器或浏览器打开该文件。你也可以复制对话框内的代码在编辑器中进行编辑。

![image-20210522174037207](/storage/images/en/schematic/file-file-source/file-file-source_20240621_172124.png)
