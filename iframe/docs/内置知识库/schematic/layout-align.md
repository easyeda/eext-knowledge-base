# 对齐

**注意**：在原理图对齐操作中，不能对符号引脚、进行对齐。

操作：

-   选择器件 - 顶部菜单 - 布局 - 对齐
-   在顶部工具栏右上角找到对齐图标

**快捷键**

-   **左对齐**：<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd>

-   **右对齐**：<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd>
-   **顶部对齐**：<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>O</kbd>
-   **底部对齐**：<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>B</kbd>

-   **左右居中**：<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd>

-   **上下居中**：<kbd>Ctrl</kbd> + <kbd>Shift </kbd>+ <kbd>H</kbd>

-   **对齐网格**：<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd>

## 左对齐

左对齐是将选中的器件中最靠左边的器件进行对齐。

![](/storage/images/en/schematic/layout-align/layout-align_20240621_172124.gif)

## 右对齐

右对齐是将选中的器件中最靠右边的器件进行对齐。

![](/storage/images/en/schematic/layout-align/layout-align_20240621_172125.gif)

## 顶部对齐

顶部对齐是将选中的器件中最顶部的器件进行对齐。

![](/storage/images/en/schematic/layout-align/layout-align_20240621_172126.gif)

## 底部对齐

底部对齐是将选中的器件中最底部的器件作为参考点进行对齐。

![](/storage/images/en/schematic/layout-align/layout-align_20240621_172127.gif)

## 左右居中

左右居中对齐是在选择的器件间找到左右两边的中间点对齐。

![](/storage/images/en/schematic/layout-align/layout-align_20240621_172128.gif)

## 上下居中

上线居中对齐是将选中的器件间找到上下的中间点进行对齐。

![](/storage/images/en/schematic/layout-align/layout-align_20240621_172129.gif)

## 对齐网格 {#网}

将选中的器件的引脚对齐在原理图设置的网格中。

![](/storage/images/en/schematic/layout-align/layout-align_20240621_172130.gif)

当勾选了选择参考对象后，再进行对齐操作时，会增加一步选择参考对象的步骤，最终对齐会根据参考对象进行对齐。
