# 器件标准化

器件标准化匹配能够帮助我们在立创商城中快速找到对应的元器件并进行物料购买。

**功能入口**：左侧面板 - 器件标准化

![图 1](/storage/images/cn/schematic/device-standardization/device-standardization_20241212_101448.png)

## 使用方法

我们常用库中的器件具有通用性，没有绑定器件供应商的信息，这时候就能在器件标准化面板中看到不匹配的红色标识。

![图 2](/storage/images/cn/schematic/device-standardization/device-standardization_20241212_101920.png)

对于不匹配的器件，我们可以选择`使用推荐器件`的`分配立创编号`的操作。

![图 3](/storage/images/cn/schematic/device-standardization/device-standardization_20241212_102207.png)

点击`使用推荐器件`操作后,能够在出现的器件管理器面板中查看推荐的目标器件，左侧面板可以批量匹配以及确定匹配选项，右侧面板可以查看目标器件以及参数对比表格。如果目标器件符合要求则可以点击`替换`。

![图 4](/storage/images/cn/schematic/device-standardization/device-standardization_20241212_103159.png)

替换后的器件如果信息完善则出现绿色标识。

![图 5](/storage/images/cn/schematic/device-standardization/device-standardization_20241212_103508.png)

如果推荐器件不符合要求则可以选择`分配立创编号`操作，在器件面板手动选择合适的器件。

![图 6](/storage/images/cn/schematic/device-standardization/device-standardization_20241212_103651.png)

对于有供应商编号但缺少其他供应商信息的器件会显示黄色标识。你也可以在顶部按钮进行`设置为不检查`，`设置为检查`的操作。

![图 7](/storage/images/cn/schematic/device-standardization/device-standardization_20241212_105243.png)
