# 旋转

注意：原理图不支持45度旋转

操作步骤：

-   .选择器件 - 布局 - 旋转
-   键盘<kbd>空格</kbd>键进行旋转

-   鼠标右键

拖动过程中进行旋转后，可以按<kbd>Esc</kbd>或单击鼠标右键，可以回退到拖动前的状态。

可以按方向键移动1个网格的距离。
