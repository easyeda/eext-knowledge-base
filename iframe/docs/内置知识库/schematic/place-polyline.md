# 折线

折线于导线不同，折线是没有电气属性的，而导线是带有电气属性的，两者虽然相同，但是实际的用法不同，折线是不能代替导线放置。

在原理图中，你可以绘制任意方向的线条。

**操作入口：**

-   顶部快捷栏 - 折线

-   顶部快捷栏 - 放置 - 折线

-   快捷键<kbd>AIT</kbd>+<kbd>L</kbd>>

![image-20221118102515310](/storage/images/cn/schematic/place-polyline/place-polyline_20240621_172124.png)

鼠标左键单击确定折线的起始点和拐点，右键结束当前折线的绘制，待确定段会消失，进入下一个折线的绘制，再次右键则退出折线绘制。

绘制过程中，如果当前有未完成的折线，按<kbd>Esc</kbd>>则清掉未完成的点，依旧保持在绘制模式，如果没有未完成的折线，按<kbd>Esc</kbd>>则退出绘制模式，光标回到选择模式。
