# 删除

在原理图中选择的元素删除

## 删除所选

操作步骤:

-   选择需要删除的对象 - 顶部菜单 - 编辑 - 删除 - 所选

-   选择需要删除的对象 - 快捷键<kbd>Delete</kbd>

![image-20210107135819433](/storage/images/en/schematic/edit-delete/edit-delete_20240621_172123.png)

## 删除对象

删除对象是可以选择需要删除的内容的，而这个删除是整体在原理图中整体删除的。

操作步骤：

-   顶部菜单 - 编辑 - 删除 - 对象

![image-20210107140532288](/storage/images/en/schematic/edit-delete/edit-delete_20240621_172124.png)

弹窗选择弹窗，勾选需要删除的对象，点击确定，即可将勾选的对象在原理图中全部删除。

![image-20210107140603622](/storage/images/en/schematic/edit-delete/edit-delete_20240621_172125.png)

## 删除全部

把原理图放置的所有元素删除。

操作步骤：

-   顶部菜单 - 编辑 - 删除 - 全部；
-   快捷键 <kbd>Ctrl</kbd> + <kbd>A</kbd> 全选 - 点击删除快捷键 <kbd>Delete</kbd>

![image-20210107141024607](/storage/images/en/schematic/edit-delete/edit-delete_20240621_172126.png)
