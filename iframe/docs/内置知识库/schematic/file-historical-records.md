# 工程历史

嘉立创EDA支持查看工程历史，用户可以实现工程与文档的历史记录查看和还原功能。

**功能入口**：

顶部菜单栏 - 文件 - 工程历史

![图 0](/storage/images/cn/schematic/historical-records/historical-records_20250305_155604.jpg)

## 使用方法

点击 `工程历史` 选项后，右侧将显示工程历史列表，用户可以在该列表中查看当前文档和工程的历史记录。

![图 1](/storage/images/cn/schematic/historical-records/historical-records_20250305_160100.jpg)

工程历史列表支持展开和收起。在查看工程历史期间，不允许对工程进行修改。

![图 2](/storage/images/cn/schematic/historical-records/historical-records_20250305_160417.jpg)

历史记录按时间段分割，列表中展示了记录的时间和最近编辑者。选中一条记录后，用户可以点击 `还原` 按钮进行记录还原。

![图 3](/storage/images/cn/schematic/historical-records/historical-records_20250305_160750.jpg)

在当前工程页面下，支持展开保存的历史记录，以原理图页或PCB图页为基本单位。

![图 5](/storage/images/cn/schematic/historical-records/historical-records_20250305_162333.jpg)

当打开的文档为 PCB 时，支持选中两条记录进行 PCB 对比。

![图 8](/storage/images/cn/schematic/historical-records/historical-records_20250305_163038.jpg)

PCB 对比时，底部框会展示对比结果，结果列表中显示差异对象及其类型，并支持点击差异项定位到具体对象位置。

![图 7](/storage/images/cn/schematic/historical-records/historical-records_20250305_163009.jpg)
