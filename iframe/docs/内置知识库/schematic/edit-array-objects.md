# 阵列对象

操作步骤：

-   选择一个对象 - 顶部菜单 - 编辑 - 阵列对象

![图 1](/storage/images/cn/schematic/edit-array-objects/edit-array-objects_20240621_172124.png)

在弹窗中输入需要阵列的行和列，还有1行距和列距，这里的行距单位默认为inch，不可修改。填写完信息后点击确定，即可将符号按照填写的数值进行阵列排序。

![图 2](/storage/images/cn/schematic/edit-array-objects/edit-array-objects_20240621_172125.png)

支持预览阵列的效果，也可以通过鼠标调整阵列的间距

![图 3](/storage/images/cn/schematic/edit-array-objects/edit-array-objects_20240621_172126.png)
