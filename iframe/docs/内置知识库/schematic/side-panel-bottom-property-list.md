# 属性清单

嘉立创EDA支持属性清单功能，用户可通过该功能以表格形式批量编辑选中对象的属性。

**功能入口**：

- 底部面板 - 属性清单

![图 0](/storage/images/cn/schematic/side-panel-bottom-property-list/side-panel-bottom-property-list_20250319_101247.jpg)

## 使用方法

在原理图中框选的对象将自动显示于属性清单的表格中。

![图 1](/storage/images/cn/schematic/side-panel-bottom-property-list/side-panel-bottom-property-list_20250319_101406.gif)

表格中可查看框选对象的详细属性信息，包括类型、名称、ID 等。支持双击编辑可修改的属性值，并提供对表格内容进行复制和粘贴的功能。

![图 2](/storage/images/cn/schematic/side-panel-bottom-property-list/side-panel-bottom-property-list_20250319_101651.jpg)

对于器件和封装属性，用户可通过点击相应单元格，在器件库中选择合适的选项。

![图 3](/storage/images/cn/schematic/side-panel-bottom-property-list/side-panel-bottom-property-list_20250319_101933.jpg)

通过右键单击表格，用户可以实现批量选中操作。

![图 4](/storage/images/cn/schematic/side-panel-bottom-property-list/side-panel-bottom-property-list_20250319_102209.jpg)

右键菜单中的`批量编辑`功能允许用户按照顺序或名称规则对表格内容进行高效修改。

![图 5](/storage/images/cn/schematic/side-panel-bottom-property-list/side-panel-bottom-property-list_20250319_102353.jpg)
