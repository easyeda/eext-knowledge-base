# 元件屏蔽

嘉立创EDA专业版支持在原理图中进行元件屏蔽，可将选中的元件不加入BOM或不转到PCB。

**操作入口：**

-   顶部菜单 - 放置 - 元件屏蔽

![图 1](/storage/images/cn/schematic/place-component-mask/place-component-mask_20240929_154918.png)

点击“元件屏蔽”后可以在弹出窗中设置屏蔽规则。
![图 2](/storage/images/cn/schematic/place-component-mask/place-component-mask_20240929_154943.png)

点击“设置样式”后可以更改屏蔽元件的外观。
![图 5](/storage/images/cn/schematic/place-component-mask/place-component-mask_20240929_155627.png)

点击“确认”后通过鼠标左键进行框选。
![图 3](/storage/images/cn/schematic/place-component-mask/place-component-mask_20240929_155050.png)

被屏蔽后的元件呈下方外观。
![图 4](/storage/images/cn/schematic/place-component-mask/place-component-mask_20240929_155113.png)

元件属性处会发生变更。
![图 6](/storage/images/cn/schematic/place-component-mask/place-component-mask_20240930_115317.png)
