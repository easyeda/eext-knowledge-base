# 左侧面板-元件

原理图编辑器界面的左侧面板与PCB界面的左侧面板功能相似，用于显示放置在原理图中的器件信息、数量和位置。
![图 1](/storage/images/cn/schematic/side-panel-left-component/side-panel-left-component_20240621_172124.png)

点击相应的器件可跳转到原理图中高亮显示出来。
