# 贝塞尔曲线 {#曲线}

**操作入口：**

-   顶部快捷栏 - 贝塞尔曲线

-   顶部菜单栏 - 放置 - 贝塞尔曲线

-   快捷键<kbd>AIT</kbd>+<kbd>Z</kbd>>

![图 0](/storage/images/cn/schematic/place-bezier/place-bezier_20240621_172124.jpg)

第一次左键确定锚点，第二次左键，确定第一个点的手柄点，第三次左键，确定第二个点的手柄点，第四次左键，确定第二个锚点，此时—段线确定，进入下—段线的绘制。

未绘制完成的线段显示为待确定段，第一次右键或<kbd>Esc</kbd>结束当前绘制，待确定段消失，只留确定段，不退出贝塞尔曲线绘制模式，再次右键或<kbd>Esc</kbd>退出绘制模式。

支持绘制过程中<kbd>Backspace</kbd>删除上—个绘制点。

绘制时支持连续2次操作在一个坐标上。

![图 2](/storage/images/cn/schematic/place-bezier/place-bezier_20240621_172125.png)

选中后可以在右侧属性修改样式。

![图 4](/storage/images/cn/schematic/place-bezier/place-bezier_20240621_172126.png)

选中时会显示贝塞尔曲线的控制点、锚点和中点。

拖动中点可以快速新增一个控制点。
