# 跨页连接标识

嘉立创EDA支持放置跨页连接标识，用于实现不同图页之间的快捷跳转与精准定位。

**功能入口**：

- 放置 - 跨页连接标识

![图 0](/storage/images/cn/schematic/place-off-page-connector/place-off-page-connector_20250313_091732.jpg)

- 顶部工具栏

![图 1](/storage/images/cn/schematic/place-off-page-connector/place-off-page-connector_20250313_091835.jpg)

跨页连接标识提供三种样式（输入、输出、双向），其功能一致，适用于不同的设计需求。

## 使用方法

跨页连接标识具有类似于网络标签的连接功能，能够表示原理图内网络的相互连接。通过相同的网络名标识，可实现网络间的连接关系。

![图 2](/storage/images/cn/schematic/place-off-page-connector/place-off-page-connector_20250313_094157.jpg)

跨页连接标识具有基于关联标识（相同网络名称）的跳转功能。用户可通过右键单击该标识，在关联项中实现图页跳转，并快速定位目标位置。

![图 3](/storage/images/cn/schematic/place-off-page-connector/place-off-page-connector_20250313_094424.gif)

支持跨图页的连接与跳转功能。

![图 4](/storage/images/cn/schematic/place-off-page-connector/place-off-page-connector_20250313_094551.gif)

在右侧属性栏中，用户可以查看当前跨页连接标识所关联的标识，并支持点击跳转。

![图 5](/storage/images/cn/schematic/place-off-page-connector/place-off-page-connector_20250313_094710.jpg)

放置跨页连接标识时，支持按 `TAB` 键，编辑标识的名称前后缀以及数值的起始值和步进，便于多次放置。

![图 6](/storage/images/cn/schematic/place-off-page-connector/place-off-page-connector_20250313_094948.jpg)
