# 非连接标识

嘉立创EDA专业版的非连接标识和网络标签类似，并不支持标准版那种独立的非连接图元。

**操作入口：**

-   顶部菜单 - 放置 - 非连接

![图 41](/storage/images/en/schematic/place-no-connect/place-no-connect_20240621_172123.png)

![图 42](/storage/images/en/schematic/place-no-connect/place-no-connect_20240621_172124.png)

非连接符号只能放置在引脚端点上。放置后是自动给符号的引脚设置属性“非连接”为是，删除非连接标识时，属性“非连接”会改为否。
