# 导入DXF

打开原理图或符号编辑器，可以支持DXF导入。

入口：

**顶部菜单 - 文件 - 导入 - DXF**

![图 1](/storage/images/cn/schematic/file-import-dxf/file-import-dxf_20240621_172124.png)

操作后选择本地需要导入的DXF文件，点击确定，会弹出导入DXF弹窗。
![图 0](/storage/images/cn/schematic/file-import-dxf/file-import-dxf_20240621_172125.png)

设置参数后，点击导入，就会进入待放置阶段，左键确认放置，右键取消放置。
