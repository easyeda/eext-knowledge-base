# 复制/粘贴

目前嘉立创EDA专业版的复制/粘贴只能应用于在同一个浏览器下进行复制/粘贴，不能跨浏览器进行复制/粘贴。

## 复制

**操作步骤**：

-   选择需要复制放内容 - 顶部菜单 - 编辑 - 复制 - 选择复制参考点

-   快捷键<kbd>Ctrl</kbd> + <kbd>C</kbd>

-   选择器件 - 按住<kbd>Ctrl</kbd> + 鼠标左键往拖动即可复制；

## 粘贴

将选择的内容复制到需要放置的地方。

操作步骤：

-   顶部菜单 - 编辑 - 粘贴

-   快捷键：<kbd>Ctrl</kbd> + <kbd>V</kbd>
