# 图片

**操作入口：**

-   顶部菜单 - 放置 - 图片

![image-20221118102247233](/storage/images/cn/schematic/place-image/place-image_20240621_172124.png)

选择好图片后，即可生成预览，单击鼠标左键放置。

![image-20201219143805329](/storage/images/en/schematic/place-image/place-image_20240621_172124.png)

选中图片即可在右侧属性面板对图片进行设置。

![image-20210518151913275](/storage/images/en/schematic/place-image/place-image_20240621_172125.png)

图片选中时，拖动控制点可以调整图片的尺寸，按住<kbd>Shift</kbd>拖动可以保持原有比例调整尺寸。

![image-20210518151934484](/storage/images/en/schematic/place-image/place-image_20240621_172126.png)

-   小技巧：图片支持按住Shift拖动进行等比例缩放。
