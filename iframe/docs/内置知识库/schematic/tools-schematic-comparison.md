# 原理图对比

嘉立创EDA提供原理图对比功能，用于展示两个原理图之间的差异。

**功能入口**：

- 顶部菜单栏 - 工具 - 原理图对比

![图 0](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250311_145354.jpg)

## 使用方法

点击`原理图对比`后，在弹出的窗口中选择需要对比的两个原理图。

![图 1](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250311_151406.jpg)

点击`开始对比`按钮后，底部面板将显示`对比结果`标签。

![图 2](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250311_152935.jpg)

中间表格用于展示对比结果，根据原理图中的图页名称进行匹配。存在差异的图页将以红色标记。

![图 3](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250311_153629.jpg)

展开表格后可以查看具体的图元类型和位号对比。完全一致的器件用黑色标记。

![图 4](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250311_153731.jpg)

存在差异的图元以红色标记。

![图 5](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250311_153822.jpg)

图页中特有图元以蓝色标记。

![图 6](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250311_153935.jpg)

支持在左侧窗口筛选对比项和只显示差异项，用户可以点击`原理图对比`重新选择原理图进行对比。

![图 7](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250311_154155.jpg)

在中间表格中，支持双击对比项切换为双窗口排列模式，便于详细对比；单击差异项可以在两个图页中进行定位。

![图 8](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250311_154657.gif)

右侧表格用于查看差异项的详细对比结果。

![图 9](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250311_155035.jpg)

支持对对比结果进行搜索、清空和导出操作。

![图 10](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250311_155107.jpg)

支持进行原理图网络比较。

![图 11](/storage/images/cn/schematic/tools-schematic-comparison/tools-schematic-comparison_20250410_111904.gif)
