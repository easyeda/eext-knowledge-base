# 视图 - 高亮网络

在原理图的设计过程中，需要经常查看网络的连接关系，可以通过顶部菜单的高亮网络功能进行网络高亮,支持高亮网络名，导线，引脚和引脚名。

**操作入口：**

-   顶部菜单 - 视图 - 高亮网络。

![图 2](/storage/images/cn/schematic/view-highlight-net/view-highlight-net_20240929_165944.png)

也可以使用快捷键<kbd> SHIFT </kbd> + <kbd> H </kbd>(快捷键配置为:嘉立创EDA专业版)或<kbd> H </kbd>(快捷键配置为：嘉立创EDA标准版)进行高亮，在高亮前需要先在画布选中你需要高亮的导线。

![图 3](/storage/images/cn/schematic/view-highlight-net/view-highlight-net_20240930_135731.png)

![图 4](/storage/images/cn/schematic/view-highlight-net/view-highlight-net_20240930_140137.png)

或者导线右键进行高亮或取消高亮。

![图 1](/storage/images/cn/schematic/view-highlight-net/view-highlight-net_20240929_165916.png)

![图 0](/storage/images/cn/schematic/view-highlight-net/view-highlight-net_20240929_165831.png)

或者在左侧网络Tab，点击导线或网络名进行高亮，但该入口的高亮是临时选中并高亮，前面的高亮菜单是持续高亮。

![图 37](/storage/images/cn/schematic/view-highlight-net/view-highlight-net_20240621_172126.png)

注意：高亮网络功能是全原理图高亮，当你高亮一个网络时，打开新的图页，图页内对应的网络的导线也会进行高亮。
