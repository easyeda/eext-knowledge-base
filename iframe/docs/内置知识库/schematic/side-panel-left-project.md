# 左侧面板-工程

为了方便打开其他工程，嘉立创EDA支持在左侧所有工程Tab直接列出工程。

![图 1](/storage/images/cn/schematic/side-panel-left-project/side-panel-left-project_20240621_172124.png)

因为目前暂不支持当前编辑器窗口同时打开多个工程，所以双击打开工程做成了在新窗口打开，双击是在当前窗口打开还是在新窗口打开可以在设置-系统-通用中进行设置。

如果需要在当前窗口打开工程（会自动关闭当前工程），可以在工程名右键菜单打开。

![图 0](/storage/images/cn/schematic/side-panel-left-project/side-panel-left-project_20240621_172125.png)

支持上下左右快捷键控制工程树，<kbd>Enetr</kbd>打开工程。

支持在切换工作区。

支持对工程、归属、文件夹进行搜索。

支持按照“工程名称”，“创建时间”，“修改时间”进行升序或降序文件夹排序。

![图 0](/storage/images/cn/schematic/side-panel-left-project/side-panel-left-project_20240930_151650.png)

在归属和文件夹上右键会显示对应的右键菜单。
