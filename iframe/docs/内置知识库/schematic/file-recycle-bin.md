# 回收站

嘉立创EDA支持回收站功能，用户可以在此查看最近90天内删除的工程、库和模块。超过90天的记录将自动清空。

**功能入口**：

- 文件 - 回收站

![图 0](/storage/images/cn/schematic/file-recycle-bin/file-recycle-bin_20250311_112523.jpg)

点击`回收站`选项后，将跳转至工作区的回收站页面。

![图 1](/storage/images/cn/schematic/file-recycle-bin/file-recycle-bin_20250311_112742.jpg)

## 工程恢复

在回收站页面中，用户可以通过`工程`标签查看最近删除的工程文件。

![图 2](/storage/images/cn/schematic/file-recycle-bin/file-recycle-bin_20250311_114248.jpg)

该页面显示了最近删除的工程标题、类型及删除时间，并支持对删除工程进行搜索。

![图 3](/storage/images/cn/schematic/file-recycle-bin/file-recycle-bin_20250311_114516.jpg)

对于已删除的工程，用户可以执行还原或彻底删除操作。此外，还支持清空回收站。请注意，彻底删除后的工程将无法再次恢复。

![图 4](/storage/images/cn/schematic/file-recycle-bin/file-recycle-bin_20250311_114631.jpg)

还原后的工程内容和位置将与删除前保持一致。

![图 5](/storage/images/cn/schematic/file-recycle-bin/file-recycle-bin_20250311_115234.jpg)

## 库恢复

在回收站页面中，用户可以通过`库`标签查看最近删除的库文件。

该页面显示了最近删除的库标题、类型及删除时间，并支持对删除库进行搜索。

![图 6](/storage/images/cn/schematic/file-recycle-bin/file-recycle-bin_20250311_135732.jpg)

对于已删除的库文件，用户可以执行还原或彻底删除操作。此外，还支持清空回收站和批量恢复库。请注意，彻底删除后的库将无法再次恢复。

![图 7](/storage/images/cn/schematic/file-recycle-bin/file-recycle-bin_20250311_135954.jpg)

## 模块恢复

在回收站页面中，用户可以通过`模块`标签查看最近删除的复用模块文件。

该页面显示了最近删除的模块标题、类型及删除时间，并支持对删除模块进行搜索。

![图 8](/storage/images/cn/schematic/file-recycle-bin/file-recycle-bin_20250311_140125.jpg)

对于已删除的模块文件，用户可以执行还原或彻底删除操作。此外，还支持清空回收站。请注意，彻底删除后的复用模块将无法再次恢复。

![图 9](/storage/images/cn/schematic/file-recycle-bin/file-recycle-bin_20250311_140219.jpg)
