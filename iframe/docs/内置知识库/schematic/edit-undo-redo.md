# 撤销/重做

能记录用户的一系列操作过程，并能根据记录恢复到当时的状态，用户在进行某个操作过后，能通过撤销回到操作之前的状态，用户在撤销之后，也能通过重做恢复前一次撤销的操作。

## 撤销

撤销当前的操作，回退当上一步的操作。

操作步骤：

-   顶部菜单 - 编辑 - 撤销

-   快捷键<kbd>Ctrl</kbd> + <kbd>Z</kbd>

![image-20211227162353191](/storage/images/en/schematic/edit-undo-redo/edit-undo-redo_20240621_172123.png)

## 重做

通过重做恢复前一次撤销的操作。

操作步骤：

-   顶部菜单 - 编辑 - 重做

-   快捷键<kbd>Ctrl</kbd> + <kbd>Y</kbd>
