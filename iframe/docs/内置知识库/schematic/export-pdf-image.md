# 导出PDF/图片

## 入口

-   **1、顶部菜单 - 文件 - 导出 - PDF/图片**
-   **2、顶部菜单 - 导出 -PDF/图片**

![图 0](/storage/images/cn/schematic/export-pdf-image/export-pdf-image_20260108_104836.png)

**文件类型**

选择导出文件的格式类型：PDF,PNG,SVG

**主题**

导出图页的配色，默认则与画布一致，黑底白图和白底黑图只有黑白两种颜色。

**线宽（只在导出PDF时可以设置）**

只有在导出PDF时可以设置，会影响1像素线的显示效果。默认则为跟随画布设置。

**尺寸**

点击右侧窗口后可选择常用尺寸或自定义尺寸。
![图 1](/storage/images/cn/schematic/export-pdf-image/export-pdf-image_20240929_163704.png)

自定义尺寸。
![图 2](/storage/images/cn/schematic/export-pdf-image/export-pdf-image_20240929_163811.png)

**菜单显示属性（只在导出PDF时可以设置）**

勾选时，在专业PDF阅读器上，可以在右键菜单中查看属性。

**仅导出图框内**

勾选时，导出的PDF只包含图纸框内的内容，而不包含图纸框外的内容。

**输出方式（只在导出PDF时可以设置）**

-   合并图页：将所选范围内的图页输出到一个PDF内。

-   单独页面：将所选范围输出的图页每一页生成一个PDF。

## 自定义水印

在导出PDF时，用户可以自定义水印内容和样式，该功能仅面向企业私有部署版本提供。

![图 1](/storage/images/cn/schematic/export-pdf-image/export-pdf-image_20250307_165255.jpg)

点击`样式设置`后，用户可以在弹窗中修改水印的内容、颜色、透明度、字体、字体大小、样式、倾斜角度和密度。
