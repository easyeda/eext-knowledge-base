# 矩形

**操作入口：**

-   顶部菜单 - 放置 - 矩形

-   快捷键 <kbd>Alt</kbd> + <kbd>R</kbd>

![image-20221118102623048](/storage/images/cn/schematic/place-rectangle/place-rectangle_20240621_172124.png)

左键单击绘图区任意位置，设置矩形端点位置，移动鼠标显示预览效果，再次左键单击结束绘制，并选中矩形。鼠标移动，左键单击确定下一个矩形的端点位置。

确认起始点后，按住<kbd>Shift</kbd>，可以绘制正方形。

如果在待确认起始点的状态下按<kbd>Esc</kbd>或鼠标右键单击则会退出矩形绘制模式，恢复鼠标状态为选择模式。

如果起始点已经确认，则按<kbd>Esc</kbd>或鼠标右键单击会取消起始点，回到待确定起始点的状态，不会退出矩形绘制状态。

![图 0](/storage/images/cn/schematic/place-rectangle/place-rectangle_20240621_172125.png)

拖动控制点可以调整矩形的尺寸，按住<kbd>Shift</kbd>拖动可以保持原有比例调整尺寸。

-   小技巧：矩形支持按住Shift拖动进行等比例缩放。
