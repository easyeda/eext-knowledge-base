# 视图缩放

缩放原理图的界面。

## 视图缩小

**操作入口：**

-   顶部工具栏 - 视图 - 缩小

-   鼠标滚轮后退缩放

![image-20221118105243665](/storage/images/cn/schematic/view-zoom/view-zoom_20240621_172124.png)

## 视图放大

**操作入口：**

-   顶部工具栏 - 视图 - 放大

-   鼠标滚轮前滚进行放大

![image-20221118105314844](/storage/images/cn/schematic/view-zoom/view-zoom_20240621_172125.png)

## 适应全部

适应原理图设计界面，跟进原理图的图页边框进行适应视图。

**操作入口：**

-   顶部工具栏 - 视图 - 适应全部

-   快捷键<kbd>K</kbd>

![image-20221118105422501](/storage/images/cn/schematic/view-zoom/view-zoom_20240621_172126.png)

## 适应选中

将选中的元素放大到合适的查看比例

**操作入口：**

-   选择器件 - 顶部菜单 - 视图 - 适应选中

-   选择器件 - 鼠标右键 - 适应选中

![图 218](/storage/images/cn/schematic/view-zoom/view-zoom_20240621_172127.png)

## 适应框选

把框选的地区放大

**操作入口：**

-   顶部菜单 - 编辑 - 适应框选，

![image-20221118105535797](/storage/images/cn/schematic/view-zoom/view-zoom_20240621_172128.png)

或顶部工具栏适应框选。

![image-20221118105604028](/storage/images/cn/schematic/view-zoom/view-zoom_20240621_172129.png)

![](/storage/images/en/schematic/view-zoom/view-zoom_20240621_172126.gif)

## 全屏

把编辑器整体按照屏幕的尺寸进行放大。

**操作入口：**

-   顶部菜单 - 视图 - 全屏

-   快捷键<kbd>F11</kbd>

![image-20221118105710954](/storage/images/cn/schematic/view-zoom/view-zoom_20240621_172130.png)
