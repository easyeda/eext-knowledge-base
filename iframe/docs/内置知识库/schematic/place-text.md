# 文本

文本能在原理图里提供文字说明，用于对器件或者原理图的说明设置。

**操作入口：**

-   顶部菜单 - 放置 - 文本

-   快捷键<kbd>Alt</kbd> + <kbd>T</kbd>

![image-20221118102848746](/storage/images/cn/schematic/place-text/place-text_20240621_172124.png)

在同一原理图内，编辑器会记住上一个文本的属性参数，在放置时自动套用上次的属性。

显示文本设置窗。

![图 0](/storage/images/cn/schematic/place-text/place-text_20240621_172125.png)

在文本框输入文字，<kbd>Ctrl</kbd>/<kbd>Shift</kbd>/<kbd>Alt</kbd>+<kbd>Enter</kbd>可以换行，点击确定或者按<kbd>Enter</kbd>即可生成预览，点击鼠标左键即可放置。

![图 1](/storage/images/cn/schematic/place-text/place-text_20240621_172126.png)

选中文本，即可在右侧的属性面板设置文本的一些属性，带“默认”则为跟随主题。

![图 3](/storage/images/cn/schematic/place-text/place-text_20240621_172127.png)
