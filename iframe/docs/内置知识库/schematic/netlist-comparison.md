# 网表对比

嘉立创EDA支持使用网表对比工具。

该工具用于帮助用户在格式转换或不同版本原理图之间进行详细的网表差异分析。该工具支持多种主流EDA软件的网表格式，确保用户能够高效地完成对比任务。

**操作入口：**

- 顶部菜单 - 工具 - 网表对比

![图 0](/storage/images/cn/schematic/netlist-comparison/netlist-comparison_20240929_161434.png)

点击“网表对比”后，左边弹出窗中可选择“网表1”和“网表2”，选择网表后点击“对比”按钮，下方会显示详细的差异表格。

![图 1](/storage/images/cn/schematic/netlist-comparison/netlist-comparison_20240929_161726.png)

支持导入本地网表文件，兼容以下格式：Allegro的TEL、Zuken的CCF、Altium的NET、PADS的ASC和TXT、以及嘉立创自身的ENET格式。

![图 3](/storage/images/cn/schematic/netlist-comparison/netlist-comparison_20250214_120036.jpg)

用户可以选择将对比结果以CSV格式导出，以便进一步分析每个网络的引脚差异。
![图 2](/storage/images/cn/schematic/netlist-comparison/netlist-comparison_20240929_161837.png)
