# 选择对象

支持多种选择方式，方便选择所需的对象。

## 全部选择

选择原理图中的全部元素。

**操作步骤**：

-   顶部菜单 - 选择对象 - 全部

-   快捷键<kbd>Ctrl</kbd> + <kbd>A</kbd>

就可以将原理图中所有的元素选中。

![图 80](/storage/images/cn/schematic/edit-select-objects/edit-select-objects_20240621_172124.png)

## 连续选择

操作步骤：

-   顶部菜单 - 编辑 - 选择对象 - 切换选择

![图 81](/storage/images/cn/schematic/edit-select-objects/edit-select-objects_20240621_172125.png)

![](/storage/images/en/schematic/edit-select-objects/edit-select-objects_20240621_172125.gif)

## 矩形内部

基本和普通的框选一样，框选一个矩形，内部的图元可以被选中

![图 82](/storage/images/cn/schematic/edit-select-objects/edit-select-objects_20240621_172126.png)

## 矩形外部

在矩形外部的元素被选中，矩形内部的则不被选中。

操作步骤

-   顶部菜单 - 编辑 - 选择对象 - 矩形外部

![图 83](/storage/images/cn/schematic/edit-select-objects/edit-select-objects_20240621_172127.png)

![](/storage/images/en/schematic/edit-select-objects/edit-select-objects_20240621_172127.gif)

## 多边形内部

在绘制的多边形图形内部的元素被选中，外部的则没有被选中。

操作步骤：

-   顶部菜单 - 编辑 - 选择对象 - 多边形内部

![图 84](/storage/images/cn/schematic/edit-select-objects/edit-select-objects_20240621_172128.png)

![](/storage/images/en/schematic/edit-select-objects/edit-select-objects_20240621_172129.gif)

## 多边形外部

在绘制的多边形图形内部的元素被不被选中，外部的则被选中。

![图 85](/storage/images/cn/schematic/edit-select-objects/edit-select-objects_20240621_172129.png)

![](/storage/images/en/schematic/edit-select-objects/edit-select-objects_20240621_172131.gif)

## 接触到线条的

接触到线条的选中。

操作步骤：

-   顶部菜单 - 编辑 - 选择对象 - 接触到线条的

![图 86](/storage/images/cn/schematic/edit-select-objects/edit-select-objects_20240621_172130.png)

![](/storage/images/en/schematic/edit-select-objects/edit-select-objects_20240621_172133.gif)

注意：当框选原理图的元件时，不会自动框选到元件在画布显示的属性。
