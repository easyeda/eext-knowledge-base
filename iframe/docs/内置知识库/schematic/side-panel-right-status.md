# 右侧面板-状态

打开文档后，右侧面板右下角区域显示当前画布和光标状态。

![image-20210518153137647](/storage/images/en/pcb/side-right-panel-status/side-right-panel-status_20240621_172123.png)

参数说明：
S (Scale)：画布缩放倍数；
G (Grid)：画布格点 X，Y 尺寸，原理图只有一个值；
X (X)：光标当前坐标 X；
Y (Y): 光标当前坐标 Y；
dX (Distance X)：当前光标在 X 轴移动的距离；
dY (Distance Y)：当前光标在 Y 轴移动的距离。
