# 分布

选择的多个器件进行等间距分布排列，，等距排列分布的的前提是要在选择两个元素以上才能进行排列分布。

## 水平等距分布

把选择的水平器件进行等距分布排列。

**操作步骤**：

-   选择需要排列分布的器件 - 顶部菜单 - 布局 - 分布 - 水平等距分布

-   选择需要排列分布的器件 - 快捷键<kbd>Ctrl</kbd> + <kbd>Shift</kbd>+ <kbd>H</kbd>

## 垂直等距分布

把选择的垂直方向器件进行等距分布排列。

操作步骤：

-   选择需要排列分布的器件 - 顶部菜单 - 布局 - 分布 - 垂直等距分布

-   选择需要排列分布的器件 - 快捷键<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd>
