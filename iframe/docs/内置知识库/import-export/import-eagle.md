# 导入EAGLE

嘉立创EDA专业版支持导入 EAGLE 6.0 版本及以上的格式文件。也支持直接导入 EAGLE 库文件。

如果是更低版本的文件请在新版本的 EAGLE 保存后重新打包导入。低于 v6.0 版本的 EAGLE 文件是加密的二进制文件，不支持直接导入。

推荐使用[嘉立创EDA文件迁移助手](/cn/import-export/easyeda-pro-format-converter)进行格式转换。

方法：

1、在 EAGLE 打开工程原理图和PCB

2、在文件菜单，另存副本为。选择 7.x 以上版本。

![图 13](/storage/images/en/import-export/import-eagle/import-eagle_20240621_172122.png)

3、找到另存为的原理图和PCB，打包为 zip 在嘉立创EDA导入。

有些 EAGLE 文件内有乱码文本，这种可能会导致导入失败，如果遇到请在新版本的 EAGLE 中打开，再另存一份后，导入。如果仍失败，请联系我们，我们会尽量修复。

导入嘉立创EDA专业版：

入口：开始页 - 快速开始 - 导入其他 - 导入 EAGLE。或者打开文件后：顶部菜单 - 文件 - 导入 - EAGLE

![图 1](/storage/images/cn/import-export/import-eagle/import-eagle_20240621_172123.png)

选择文件导入

![图 2](/storage/images/cn/import-export/import-eagle/import-eagle_20240621_172124.png)
