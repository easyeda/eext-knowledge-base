# 导入Diptrace

嘉立创EDA专业版不支持直接导入Diptrace格式文件。但可以通过Diptrace导出其他EDA格式后再导入。

方法：

1、打开Diptrace的原理图或PCB文件

2、在顶部菜单 - 文件 - 导出，导出EAGLE格式文件。
![picture 1](/storage/images/en/import-export/import-diptrace/import-diptrace_20240621_172122.png)

3、在嘉立创EDA选择EAGLE文件进行导入即可。
![picture 2](/storage/images/en/import-export/import-diptrace/import-diptrace_20240621_172123.png)

库文件同理

注意：

1、因图元功能和设计不一样，无法确保导入前后完全一致，请仔细检查。
