# 导入Sprint Layout

嘉立创EDA专业版暂不支持直接导入Sprint Layout格式文件。但可以通过其他EDA工具转一次再导入。

方法：

1、使用Diptrace，导入Sprint Layout文件
下载diptrace：https://www.diptrace.com/download/download-diptrace/

2、打开PCB模块，选择Sprint Layout的PCB文件导入，在顶部菜单 - 导入菜单。
![picture 5](/storage/images/en/import-export/import-sprint-layout/import-sprint-layout_20240621_172122.png)
![picture 6](/storage/images/en/import-export/import-sprint-layout/import-sprint-layout_20240621_172123.png)

3、导入后，在文件菜单 - 导出 - Eagle Board文件

![picture 7](/storage/images/en/import-export/import-sprint-layout/import-sprint-layout_20240621_172124.png)

4、再导入Eagle文件到嘉立创EDA中即可

注意：

1、因图元功能和设计不一样，无法确保导入前后完全一致，请仔细检查。
