# 导入CADStar

嘉立创EDA专业版暂不支持直接导入CADStar格式文件。但可以使用其他EDA工具转一次后在导入。

方法：

1、下载安装Pulsonix并打开。

2、在顶部菜单 - 文件 - 数据转换。
![picture 8](/storage/images/en/import-export/import-cadstar/import-cadstar_20240621_172122.png)
选择设计文件导入：
![picture 10](/storage/images/en/import-export/import-cadstar/import-cadstar_20240621_172123.png)
选择CADStar文件：
![picture 15](/storage/images/en/import-export/import-cadstar/import-cadstar_20240621_172124.png)
设置需要导入的文件所在目录，和输出文件所在目录，后进行转换：
![picture 12](/storage/images/en/import-export/import-cadstar/import-cadstar_20240621_172125.png)

3、导入后，在顶部文件菜单 - 文件 - 另存为，选择 PADS 格式保存。
![picture 14](/storage/images/en/import-export/import-cadstar/import-cadstar_20240621_172126.png)

4、在嘉立创EDA选择 PADS 文件类型进行导入即可。

注意：

1、因图元功能和设计不一样，无法确保导入前后完全一致，请仔细检查。
