# 导出嘉立创EDA(专业版)

嘉立创EDA专业版支持把工程文件另存到云端服务器或者本地文件中。

不支持导出嘉立创EDA标准版格式。

## 导出流程

-   顶部菜单 - 文件 - 另存为

![图 1](/storage/images/cn/import-export/export-lceda-pro/export-lceda-pro_20240621_172123.jpg)

### 工程另存为（云端）

选择工程另存为（云端）会打开另存为的窗口，可以在这里设置工程所属、工程名称、工程链接、描述、工程协作模式等等内容
![图 2](/storage/images/cn/import-export/export-lceda-pro/export-lceda-pro_20240621_172124.jpg)

设置完成后点击保存即可另存到云端

### 工程另存为（本地）

选择此功能可以将工程文件保存到本地

![图 3](/storage/images/cn/import-export/export-lceda-pro/export-lceda-pro_20240621_172125.jpg)

可以在这里选择是否要加密，勾选后即可开始设置密码，需要输入两次相同的密码然后点击确认即可加密导出

![图 5](/storage/images/cn/import-export/export-lceda-pro/export-lceda-pro_20240621_172126.jpg)

### 文档另存为（云端）

选择此菜单可以单独将目前页面打开的文档另存到云端

![图 6](/storage/images/cn/import-export/export-lceda-pro/export-lceda-pro_20240621_172127.jpg)

### 文档另存为（本地）

选择此菜单可以将当前打开的文档单独保存到本地，也支持进行加密保存

![图 7](/storage/images/cn/import-export/export-lceda-pro/export-lceda-pro_20240621_172128.jpg)

### 复用模块另存为（云端）

选择此菜单可以将当前工程的原理图和PCB一起另存为一个复用模块，以方便下次直接调用此电路

![图 9](/storage/images/cn/import-export/export-lceda-pro/export-lceda-pro_20240621_172129.jpg)

在弹窗中你需要分别在工程中选择一个模板原理图和模板PCB，然后设置一个名称，以及所有者、分类、描述等相关内容，然后点击确认即可另存为一个复用模块

![图 10](/storage/images/cn/import-export/export-lceda-pro/export-lceda-pro_20240621_172130.jpg)

### 复用模块另存为（本地）

选择将原理图和PCB另存为复用模块并且保存到本地，在这个窗口中你也需要选择复用模块的模板原理图和PCB，以及复用模块名称。

![图 11](/storage/images/cn/import-export/export-lceda-pro/export-lceda-pro_20240621_172131.jpg)

你可以选择是否要设置密码

![图 12](/storage/images/cn/import-export/export-lceda-pro/export-lceda-pro_20240621_172132.jpg)

点击点击确认即可将工程文件保存为复用模块并下载到本地

在编辑器开始页以及顶部菜单可以导入保存在本地的工程压缩包。

![图 13](/storage/images/cn/import-export/export-lceda-pro/export-lceda-pro_20240621_172133.jpg)
