# 导入Protel

嘉立创EDA专业版支持导入 Protel。

入口：开始页 - 导入其他，选择导入 Protel。

![图 6](/storage/images/cn/import-export/import-protel/import-protel_20240621_172123.png)

目前仅支持导入 ASCII 格式的 Protel 99 SE 文件，导入前请先使用 Protel 导出为 ASCII 格式。

步骤：

1、Protel 99SE 打开工程，打开原理图或PCB

2、在顶部文件菜单，另存副本为 ASCII 文件

![图 1](/storage/images/en/import-export/import-protel/import-protel_20240621_172123.png)

3、在左侧列表找到刚刚另存为的文件，右键导出到本地文件夹

![图 2](/storage/images/en/import-export/import-protel/import-protel_20240621_172124.png)

4、找到导出的文件后，把原理图和PCB压缩为 zip 文件，再导入嘉立创EDA专业版。

如果是更低版本的 Protel 文件，可能无法正常导入，建议先在 Protel 99 SE 重新导出一次。

注意：

1、请把原理图和PCB压缩为 ZIP 一起导入，否则单独导入原理图将会没有封装关联。
2、文本等图元有轻微差异，注意仔细检查。
