# 导入Pulsonix

嘉立创EDA专业版暂不支持直接导入Pulsonix格式文件。但可以先另存为其他格式后在导入。

方法：

1、使用Pulsonix打开原理图或PCB

2、在顶部菜单 - 文件 - 另存为。选择Orcad或PADS格式文件。
![picture 3](/storage/images/en/import-export/import-pulsonix/import-pulsonix_20240621_172122.png)
![picture 4](/storage/images/en/import-export/import-pulsonix/import-pulsonix_20240621_172123.png)

3、在嘉立创EDA选择对应的文件类型进行导入即可，Orcad或PADS文件导入。

注意：

1、因图元功能和设计不一样，无法确保导入前后完全一致，请仔细检查。
