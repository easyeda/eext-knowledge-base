# 十三 QC

&emsp;&emsp;QC指质量检测。主要通过人工进行检测，主要检测板子外观和数量是否达标。板子外观方面，主要看是否有明显的缺陷，比如划伤、字符模糊等，还需要检测需要V割的板子是否有V割过。
<br/>
&emsp;&emsp;在电路板上的某个地方，会有我们的客户编号，这个是嘉立创的工程人员在制作MI文件的时候，给我们添加到电路板上的，以便分辨板子，不加客户编号会使找板变得极其麻烦。负责锣边点数的工作人员拿到电路板后，找到客户编号，输入到ERP系统中，就可以打印出标签。图13-1所示为嘉立创的QC车间。
