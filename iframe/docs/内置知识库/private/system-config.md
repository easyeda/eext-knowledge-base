# 私有部署系统配置要求

## 服务机配置要求

使用人数 100 人内

| 配置项 | 配置要求                                                                                                                   |
| ------ | -------------------------------------------------------------------------------------------------------------------------- |
| CPU    | Intel 处理器 4 核心及以上，推荐 i3-9100 及以上配置                                                                         |
| 架构   | X86、AMD64                                                                                                                 |
| 内存   | 16G DDR4-2666 内存及以上                                                                                                   |
| 磁盘   | 系统盘至少 128G SSD<br>数据盘至少 100G SSD，HDD 亦可但可能部分功能性能表现不理想                                           |
| 系统   | Windows：Windows Server 2016 或 Windows 10 及以上系统<br>Linux：CentOS-7.4 或 Ubuntu-20.04 等主流操作系统，内核 4.9 及以上 |

## 用户机配置要求

| 配置项 | 配置要求                                                                                                                                                                            |
| ------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| CPU    | 2 核心及以上，推荐 i3-9100 及以上配置，支持国产 CPU                                                                                                                                 |
| 架构   | X86、AMD64、LoongArch64、SW_64、ARM64                                                                                                                                               |
| 内存   | 1 万 Pin 以内 8G DDR4 内存及以上，1 万 Pin 以上推荐 16GB 及以上                                                                                                                     |
| 显卡   | 推荐 2017 年后的独立显卡，推荐 GTX1080 及以上；如果只有集显建议是 2019 年后的 CPU（AMD64、X86）；<br>国产 CPU（SW64、LoongArch64等） 时需要独立显卡；<br>建议显卡驱动升级到最新版本 |
| 磁盘   | 系统盘至少 128G SSD，数据盘至少 100G HDD，推荐 SSD                                                                                                                                  |
| 浏览器 | Chrome、Edge、Firefox 最新版。<br>推荐 Chrome，至少 V120 版及以上，64 位版本，并开启硬件加速。国产浏览器内核版本太低不支持                                                          |
| 系统   | Windows：Windows 10 及以上系统，不推荐 Windows 7 ，不兼容 Windows XP<br>Linux：Linux 内核 4.9 及以上，支持 Ubuntu、UOS、deepin、Kylin 等。<br>Mac：MacOS 10.12 及以上               |
