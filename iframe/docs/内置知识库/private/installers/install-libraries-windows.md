# 安装系统库包（Windows）

需先准备 常用库 或 系统库 文件

1. 服务器桌面右下角，任务栏中，在私有部署图标右键，打开设置

![打开私有化系统设置](/storage/images/cn/private/installers/import-syslib-windows/import-syslib_20240626_173853.png)

1. 点击 更新库 右侧的文件夹图标，选择库压缩包

![点击文件夹图标选择库包](/storage/images/cn/private/installers/import-syslib-windows/import-syslib_20240626_173903.png)

1. 点击文件夹图标右侧的【导入】 按钮，会提示停止服务，点击【确认】按钮开始导入

![点击导入按钮会提示停止服务](/storage/images/cn/private/installers/import-syslib-windows/import-syslib_20240626_173920.png)

1. 导入过程会暂停服务，同时弹出控制台黑窗口显示导入信息

![导入过程会暂停服务](/storage/images/cn/private/installers/import-syslib-windows/import-syslib_20240626_173942.png)

![导入过程会弹出控制台黑窗口](/storage/images/cn/private/installers/import-syslib-windows/import-syslib_20240626_174003.png)

1. 导入完成后会自动启动服务

![导入完成后会自动启动服务](/storage/images/cn/private/installers/import-syslib-windows/import-syslib_20240626_174024.png)
