# 嘉立创EDA私有部署

## 什么是私有部署？

嘉立创 EDA 私有部署方案，是将嘉立创 EDA 整个系统部署到企业内部私有服务器，数据与外界互联网完全隔离，只在企业的局域网内部使用，以确保数据安全。

## 私有部署特点

### 数据安全与管控

-   私有部署安装在企业私有服务器，与外界互联网隔离
-   提供企业级的账号、工程、团队、权限、角色管理
-   本地与服务器、手动与自动备份等多种备份方式

### 团队协作

-   跨平台、跨设备应用、支持多种操作系统，支持多人同时设计同一个工程
-   精细的权限管理，完备详尽的日志系统
-   内置评审功能与聊天工具，免拉群讨论设计方案

### 扩展插件及API

-   支持扩展插件接口，可自行开发插件脚本
-   支持对接 ERP 与 PLM 系统，使用公司内部库数据
-   支持生产文件一键上传、一键下单，一键生产制造

## 私有部署优势

### 版权优势

-   拥有独立自主知识产权，避免被国外厂商“卡脖子”
-   自研渲染引擎、布线引擎，自研铺铜、飞线算法
-   软件国产化，正版化，无版权困扰，避免被打盗版

### 价格优势

-   不到国外软件十分之一的价格
-   包含部署安装、使用培训、技术支持及升级服务

### 服务优势

-   专属客户经理，专属服务群，专属技术支持
-   研发团队扎根本土，可快速解决用户问题
-   协助用户存量数据迁移
-   支持定制化域名，支持 API 改造与定制化开发
