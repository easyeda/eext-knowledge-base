# 原理图设计

“原理图”（Schematic Diagram）是一种图形化表示方法，用于展示电路的组成元件及其相互连接的方式。原理图主要展示了电路的功能和工作原理。

## 一、界面介绍与基本设置

新建的工程默认会创建一块板子，板子自带一个原理图和一个 PCB
双击原理图即可打开图页

## 三、放置器件

### 1. 放置器件的三种方式

-   从常用库放置

-   从器件弹窗放置

-   从底部库面板放置

### 2. 放置器件的操作

点击 放置 → 器件，搜索您需要的器件即可放置

搜索出想要的元件，点击 放置 按钮即可放置在画布

鼠标滚轮缩放画布，按住鼠标右键拖拽移动画布；空格键旋转元件方向，鼠标单击放置在画布，鼠标右击取消放置

选中图元后，鼠标拖动可移动图元位置，按 Delete 键可删除图元，双击可修改内容

## 五、放置电源符号和网络标签

### 1. 放置网络标识

点击 放置 → 网络标识，可选用合适的网络标识放置，也可通过工具栏快速放置
双击网络标识，可对网络标识进行修改

点击 放置 → 网络标签（或工具栏上的快捷方式），可放置网络标签
双击网络标签，可对网络标签进行修改

### 2. 扇出网络标识

## 五、更新到PCB

点击 设计 → 更新/转换原理图到PCB 即可将已经绘制好的原理图更新到 PCB
更新后核对更新信息，并点击 确认修改
