# 工作区

工作区是个管理系统，在工作区可以管理您的团队、工程等

## 团队

1. 新建团队
   在 团队 页面，点击【新建团队】按钮，即可进入新建团队页面
   填写团队信息后，即可新建团队
   [图片]
   [图片]

2. 添加团队成员
   在团队详情页，点击【添加团队成员】按钮，即可添加团队成员
   [图片]

3. 移除团队成员
   在团队详情页，点击【移除】按钮，即可移除团队成员，也可以点击【批量移除成员】按钮一次性移除多名成员
   [图片]

4. 解散团队
   点击右上角的【解散团队】按钮，即可解散团队
   [图片]

## 工程

1. 新建工程
   新建工程前，请确认自己是否有团队，需要有团队才能新建工程
   如没有团队，请新建团队，或让其他人邀请你加入团队

在 工程 页面，点击【新建工程】按钮，即可进入新建工程页面
填写工程信息后，即可新建工程
[图片]
[图片]

2. 添加工程成员
   在工程详情页，成员选项卡中，点击【添加成员】按钮，即可添加成员
   可以选择从团队中添加成员，也可以选择从企业中添加成员
   [图片]

3. 打开工程编辑
   在工程详情页面，点击【编辑器打开】按钮，即可在编辑器打开工程
   [图片]

4. 删除工程
   点击右上角的【删除】按钮，即可删除工程，请确保删除时无人正在编辑此工程
   [图片]

## 账户

1. 修改密码
   点击 账户 进入账户页，点击左上角【修改登录密码】按钮，在弹窗中修改登录密码。
