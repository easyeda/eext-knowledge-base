# 阵列对象

器件阵列和封装库的阵列功能相似，只是在封装库的基础上多加了一些的功能。

**操作入口**：

-   选择器件 - 顶部菜单 - 编辑 - 阵列对象

![image-20211223133808307](/storage/images/cn/pcb/edit-array-objects/edit-array-objects_20240621_172124.png)

弹窗阵列的属性面板，默认是矩形的属性面板。

![](/storage/images/cn/pcb/edit-array-objects/edit-array-objects_20240621_172125.png)

-   **类型**：可选择阵列的形状
-   **位置**：矩形阵列的位置是默认为选中器件的位置，而圆形阵列则需要自己选择。

-   **分布类型**：能选择先水平后垂直或先垂直后水平的阵列排序设置。

-   **每行数量**：这种阵列行的数量。

-   **每列数量**：设置阵列写列的数量。

-   **列距**：设置列阵列之间的距离。

-   **行距**：设置阵列行的距离。
-   **元素旋转的角度**：设置阵列后摆放的角度。

圆形阵列的属性面板

![](/storage/images/cn/pcb/edit-array-objects/edit-array-objects_20240621_172126.png)

-   **位置**：设置圆形阵列的中心点。
-   **方向**：可设置阵列的方向为逆时针阵列还是顺时针阵列。

-   **起始旋转角度**：选择位置后系统就会默认起始的角度了。

-   **起始旋转角度**：设置起始阵列对象的旋转角度。

-   **数量**：设置阵列对象的数量。
-   **间隔角度**：设置每个阵列对象的间距。

-   **半径**：默认和选择的半径，不可更改。

-   **选择对象是否适合圆形**：选择器件是否适合圆形阵列的使用。

## 矩形阵列设置

选择矩形，阵列的起点默认为选中的器件位置。

![](/storage/images/cn/pcb/edit-array-objects/edit-array-objects_20240621_172127.png)

分布类型我们可以选择先水平后垂直或先垂直后水平，根据个人需求选择。

![](/storage/images/cn/pcb/edit-array-objects/edit-array-objects_20240621_172128.png)

设置阵列的属性，旋转的角度先不设置，点击确定生成阵列。

![](/storage/images/cn/pcb/edit-array-objects/edit-array-objects_20240621_172129.png)

没有选择角度的阵列

![](/storage/images/cn/pcb/edit-array-objects/edit-array-objects_20240621_172130.png)

设置旋转阵列

![](/storage/images/cn/pcb/edit-array-objects/edit-array-objects_20240621_172131.png)

阵列旋转，阵列的本体是不会随着阵列生成旋转的。
