# 导出3D文件

嘉立创EDA专业版目前只支持导出两种格式的 3D 文件。

**导出方法**：

-   顶部菜单 - 导出 - 3D文件

![image-20211223141221768](/storage/images/cn/pcb/export-3d-file/export-3d-file_20240621_172124.png)

![图 3](/storage/images/cn/pcb/export-3d-file/export-3d-file_20240621_172125.jpg)

**文件类型**

-   **STEP**：导出 STEP 格式必须是 PCB 器件绑定的 3D 模型也是 STEP 格式的，否则导出的 STEP 将不带模型。

-   **OBJ**：导出 OBJ 格式是一个压缩包，里面包含材质文件。可以使用 windows10 自带的 3D 查看器预览。

**导出对象**

-   **PCB**：PCB基板，里面包含挖槽区域

-   **元件模型**：元件的3D模型

-   **过孔**：过孔和多层焊盘的金属化钻孔

-   **丝印**：顶面和底面丝印层的文本和图形

-   **信号层线路**：顶层和底层的铜箔

**为未绑定3D模型的元件自动生成3D模型（根据“高度”属性）**：勾选后，没有绑定3D模型的元件，但有Height属性时，将根据元件的矩形形状和高度信息生成一个长方体来导出

![图 1](/storage/images/cn/pcb/export-3d-file/export-3d-file_20240621_172126.jpg)

**元件未成功加载提示**

在导出过程中，如果勾选了元件模型，而部分元件模型未成功加载出来，将会首先出现提示弹窗

![图 2](/storage/images/cn/pcb/export-3d-file/export-3d-file_20240621_172127.jpg)

**_失败原因类型_**

-   未绑定模型

-   未绑定STEP格式的3D模型

-   绑定的3D模型id未找到

-   网络原因未成功加载3D模型

在处理完异常后点击重新加载全部模型按钮，然后点击处理完成，继续导出将最终导出3D文件到本地。

导出的3D文件可以在在线3D查看器进行预览效果：https://viewer.forface3d.com/online
