# 移除回路 {移}

如果你需要在布线过程中，自动移除导线环路进行修正布线，你需要开启移除回路功能。

**操作入口：**

-   顶部菜单栏 - 布线 - 移除回路

![图 0](/storage/images/cn/pcb/route-remove-loop/route-remove-loop_20240621_172124.jpg)

或布线过程中右键菜单（默认按CTRL+右键）

![图 63](/storage/images/cn/pcb/route-remove-loop/route-remove-loop_20240621_172125.png)

或画布右侧属性面板

![图 64](/storage/images/cn/pcb/route-remove-loop/route-remove-loop_20240621_172126.png)

当移除回路打开时，绘制的导线检测到有回路时，会自动把上一个回路消除，减少手动删除的操作。移除回路会自动删除过孔。

![图 1](/storage/images/en/pcb/route-remove-loop/route-remove-loop_20240621_172123.png)
