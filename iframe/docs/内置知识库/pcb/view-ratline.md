# 飞线

飞线是基于相同网络产生。

当两个封装的焊盘网络相同时，会出现飞线，表示这两个焊盘可以通过导线连接。嘉立创EDA专业版支持器件飞线的隐藏和显示

![image-20210520164414523](/storage/images/en/pcb/view-ratline/view-ratline_20240621_172123.png)

**未隐藏飞线前**

## 隐藏

-   **顶部菜单 - 视图 - 飞线 - 隐藏全部**

![image-20210520164627401](/storage/images/en/pcb/view-ratline/view-ratline_20240621_172124.png)

![image-20210520164703030](/storage/images/en/pcb/view-ratline/view-ratline_20240621_172125.png)

## 显示

-   **顶部菜单 - 视图 - 飞线 - 显示**，即可把隐藏的飞旋显示出来。

![image-20210520165606448](/storage/images/en/pcb/view-ratline/view-ratline_20240621_172126.png)

## 单器件飞线隐藏

-   **选中器件 - 顶部菜单 - 飞线 - 隐藏所选**，点击后，所选器件的飞线会单独隐藏。
