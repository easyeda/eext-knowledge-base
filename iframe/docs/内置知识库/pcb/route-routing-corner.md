# 布线角度

在布线时候切换走线的角度，支持45°、90°、圆弧45°、圆弧90°、圆弧自由角度、线条自由角度。

**操作方法**：

-   顶部菜单 - 布线 - 布线拐角
-   顶部工具栏布线拐角去快速切换拐角模式
-   在布线模式通过快捷键<kbd>L</kbd>改变布线的角度
