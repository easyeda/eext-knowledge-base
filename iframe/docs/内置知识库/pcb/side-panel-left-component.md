# 左侧面板-元件

左侧面板元件界面是显示放置在当前PCB界面的器件数量、焊盘、位号、封装等信息。

单击可高亮选择的元素，双击可追随到PCB界面并且高亮放大。

![image-20210105171034480](/storage/images/en/pcb/side-left-panel-component/side-left-panel-component_20240621_172123.png)

## 位号

显示在PCB界面同一类型的位号器件

点击位号或者封装，即可跳转到PCB界面对应的位号下的器件，点击器件下的焊盘即可高亮选中器件的焊盘。

![image-20210105173847021](/storage/images/en/pcb/side-left-panel-component/side-left-panel-component_20240621_172124.png)

## 器件

显示放置在PCB界面的器件，不同器件的名称将会分开显示。

点击器件，即可跳转到PCB界面对应的器件，点击器件下的焊盘即可高亮选中器件的焊盘。

![image-20210105174056214](/storage/images/en/pcb/side-left-panel-component/side-left-panel-component_20240621_172125.png)

## 封装

显示放弃PCB界面放置的封装，不同的封装将会分类显示。

相同，点击封装，即可跳转到PCB界面对应的封装，点击器件下的焊盘即可高亮选中器件的焊盘。

![image-20210105174212685](/storage/images/en/pcb/side-left-panel-component/side-left-panel-component_20240621_172126.png)
