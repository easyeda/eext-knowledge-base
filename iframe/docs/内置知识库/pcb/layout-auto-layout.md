# 自动布局

嘉立创EDA支持简单的自动布局功能，目前仅是体验阶段（预览版），会不断迭代更新。

**操作入口：**

-   顶部菜单 - 布局 - 自动布局

![图 176](/storage/images/cn/pcb/layout-auto-layout/layout-auto-layout_20240621_172124.png)

当后续开发完成后会根据原理图的元件位置，和元件的连接关系，根据PCB板子的大小，间距规则等自动布局。

如果有绘制板框，自动布局会根据板框自动摆放元件，如果没有绘制板框，自动布局将自动生成一个最优的矩形边框，并自动布局。

**注意**：自动布局会根据内置的规则进行布局，并不能完全替换人工，在自动布局后请注意检查是否符合设计要求，可能需要进行手动修正布局。
