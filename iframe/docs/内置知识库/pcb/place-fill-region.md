# 填充区域

嘉立创EDA提供了一个填充区域。你可以绘制所需的填充信息，该功能与铺铜有类似的地方，但是实心填充不能与不同网络的元素产生间隙。

- 顶部菜单 - 填充区域

![image-20210519111452763](/storage/images/cn/pcb/place-fill-region/place-fill-region_20240621_172124.png)

绘制完之后，自动为填充区域设置接触到网络，可以通过出现的弹窗修改填充区域的层和网络。

![图 0](/storage/images/cn/pcb/place-fill-region/place-fill-region_20240621_172125.jpg)

**属性**

- **层**：支持将实心填充切换至其他层：顶层，底层，顶层丝印，底层丝印，文档，多层等，这些层需要在层工具开启后才会全部显示出来。
- **网络**：在顶层和底层，或其他内层信号层时，可以对其设置网络使其具有电气特性。如果使用实心填充直接连接两个焊盘，需要将它们的网络设为一样，实心填充需要盖过焊盘中心，并且需要用单个实心填充连接起来，否则飞线不会消失。
- **类型**： 修改填充的区域，默认在填充区域层，不推荐改成其他层。默认填充区域层即可。
- **编辑坐标点**：支持实心填充坐标点编辑，编辑出你所需要的形状，包括圆弧的形状。

# 桥接铜

嘉立创EDA支持桥接铜功能，该功能用于实现 PCB 中不同网络之间的短接操作。

**功能入口**：

- 在填充区域右侧的属性栏中找到“桥接”属性，然后点击“桥接铜”选项。

![图 0](/storage/images/cn/pcb/place-fill-region/place-fill-region_20250324_173352.jpg)

用户可在“桥接铜”选项下方进行桥接铜网络的添加与删除操作。启用桥接铜功能后，填充区域的网络将进入不可编辑状态。

![图 1](/storage/images/cn/pcb/place-fill-region/place-fill-region_20250326_093210.jpg)

设置了桥接铜网络后，列表内的不同网络短接，注意相同网络不具有连接关系。图中 AGND 和 GND 被短接。

![图 2](/storage/images/cn/pcb/place-fill-region/place-fill-region_20250326_093554.jpg)

桥接铜的区域需仅具有列表添加的所有网络，否则会出现 DRC 报错。

![图 4](/storage/images/cn/pcb/place-fill-region/place-fill-region_20250326_094111.jpg)

设置好的桥接铜区域，支持查找，查找中种类选择填充区域，可在下方找到桥接铜和桥接网络的搜索属性。

![图 5](/storage/images/cn/pcb/place-fill-region/place-fill-region_20250326_094308.jpg)
