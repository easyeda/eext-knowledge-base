# 移动

把选择的元素进行移动。

**操作入口**：

-   选择器件顶部菜单 - 编辑 - 移动或选择器件点击鼠标右键

![image-20210522141428193](/storage/images/cn/pcb/edit-move/edit-move_20240621_172124.png)

## 跟进中心移动

-   选择器件 - 顶部菜单 - 编辑 - 移动 - 根据原点移动

-   选择器件 - 点击鼠标右键 - 移动 - 根据中心移动

-   快捷键<kbd>M</kbd>

![image-20210105153353713](/storage/images/cn/pcb/edit-move/edit-move_20240621_172125.png)

## 根据原点移动

-   选择器件 - 顶部菜单 - 编辑 - 移动 - 根据原点移动
-   鼠标右键 - 移动 - 根据原点移动

选择后即可将器件根据封装的原点进行拖动

![image-20210522141255352](/storage/images/cn/pcb/edit-move/edit-move_20240621_172126.png)

![](/storage/images/cn/pcb/edit-move/edit-move_20240621_172127.gif)

## 根据参考点移动 {#移动aru}

将选择的器件根据用户选择的参考点移动，点击后选择一个作为移动的参考点来进行拖动。

-   选择器件 - 顶部菜单 - 移动 - 根据参考点移动
-   鼠标右键 - 移动 - 根据参考点移动
