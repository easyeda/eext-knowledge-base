# 焊接辅助工具

嘉立创EDA专业版提供一个简单的焊接帮助工具，方便在焊接的时候进行元件定位。

**操作入口：**

-   工具 - 焊接辅助工具

![图 15](/storage/images/en/pcb/tools-soldering-assistance-tool/tools-soldering-assistance-tool_20240621_172123.png)

![图 16](/storage/images/en/pcb/tools-soldering-assistance-tool/tools-soldering-assistance-tool_20240621_172124.png)

在左侧元件列表进行勾选已焊接部分，支持元件位号聚合或不聚合。移动鼠标经过元件列表时，右侧的3D预览会同时高亮元件的模型，方便定位。

顶部支持按分类显示全部元件，还是显示需要焊接的元件等。
