# 锁定/解锁

将选中的元素锁定后，就不可直接移动或删除。点击需要锁定的元件，鼠标右键选择锁定或在右侧属性面板中将**锁定**属性右侧的下拉框切换为是，即可把元件锁定。

![image-20210522140020900](/storage/images/cn/pcb/edit-lock-unlock/edit-lock-unlock_20240621_172124.png)

![image-20210522140054386](/storage/images/cn/pcb/edit-lock-unlock/edit-lock-unlock_20240621_172125.png)

单图元锁定：在其属性的锁定属性设置为是。

批量锁定：在右侧过滤面板，直接把对应图元或者状态的对象进行取消勾选即可锁定；

![图 1](/storage/images/cn/pcb/edit-lock-unlock/edit-lock-unlock_20240621_172126.png)

在图层面板，把对应的层进行锁定也可以批量锁定。

![图 2](/storage/images/cn/pcb/edit-lock-unlock/edit-lock-unlock_20240621_172127.png)
