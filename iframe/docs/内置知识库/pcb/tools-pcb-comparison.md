# PCB 对比

嘉立创EDA提供了 PCB 对比工具，用于快速分析不同 PCB 之间的差异，能够帮助用户高效识别和定位设计变更。

**操作入口**：

顶部菜单栏 - 工具 - PCB 对比...

![图 0](/storage/images/cn/pcb/tools-pcb-comparison/tools-pcb-comparison_20250111_143817.jpg)

在" PCB1 "和" PCB2 "的下拉框中选择需要分析的 PCB，点击“开始对比”按钮后，工具会按照输入的对比误差和对比尺寸进行 PCB 对比。

![图 1](/storage/images/cn/pcb/tools-pcb-comparison/tools-pcb-comparison_20250111_143844.jpg)

用于对比的 PCB 会打开为两个分页进行展示，底部栏增加“对比结果”的可关闭窗口，可以点击下方列表进行差异定位。

![图 5](/storage/images/cn/pcb/tools-pcb-comparison/tools-pcb-comparison_20250111_152256.jpg)

左侧列表展示 PCB 的差异类型及其所在图层，用户可以根据需要筛选显示类型，并选择是否显示特定差异。

![图 6](/storage/images/cn/pcb/tools-pcb-comparison/tools-pcb-comparison_20250111_152336.jpg)

右侧面板展示每个差异的具体信息,可以查看对比对象的具体差异类型。

![图 4](/storage/images/cn/pcb/tools-pcb-comparison/tools-pcb-comparison_20250111_150632.jpg)
