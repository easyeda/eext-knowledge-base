# 堆叠表

嘉立创EDA专业版支持放置 PCB 层堆叠的表格，展示当前设计。

**放置方法**：

-   顶部菜单 - 放置 - 堆叠表

![图 9](/storage/images/cn/pcb/place-stack-table/place-stack-table_20240621_172124.png)

设置堆叠表的参数：

![图 10](/storage/images/cn/pcb/place-stack-table/place-stack-table_20240621_172125.png)

点确定后选择堆叠表左上角顶点的位置，堆叠表默认放在文档层。

![图 11](/storage/images/cn/pcb/place-stack-table/place-stack-table_20240621_172126.png)

放置后的堆叠表是多种图元组合而成，你可以解散组合成单独的图元编辑。如果图层管理器有调整图层参数，需要删除并重新放置堆叠表，堆叠表不会自动更新
