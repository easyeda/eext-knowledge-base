# 特殊复制

嘉立创EDA专业版支持特殊复制功能，用于实现PCB板框按指定距离或比例缩放。

**操作入口**：

-   选择器件顶部菜单 - 编辑 - 特殊复制

![图 1](/storage/images/cn/pcb/edit-copy-special/edit-copy-special_20240930_105733.png)

## 使用方法

首先在右侧属性栏选择复制选项,支持内缩,外扩,按指定比例缩放。

![图 2](/storage/images/cn/pcb/edit-copy-special/edit-copy-special_20240930_105940.png)

然后鼠标左键单击PCB的板框，即可生成修改后的板框。

![图 3](/storage/images/cn/pcb/edit-copy-special/edit-copy-special_20240930_110113.png)

修改后的板框图。
