# 导出ODB++

嘉立创EDA专业版目前支持导出ODB++ v8版本格式

**操作入口**:

- 顶部菜单 - 文件 - 导出 - ODB++
- 顶部菜单 - 导出 - ODB++

![图 7](/storage/images/cn/pcb/export-odb-plus-plus/export-odb-plus-plus_20250213_091925.jpg)

![图 6](/storage/images/cn/pcb/export-odb-plus-plus/export-odb-plus-plus_20250213_091839.jpg)

打开窗口后直接点击**导出ODB**++按钮即可导出ODB++文件

![图 8](/storage/images/cn/pcb/export-odb-plus-plus/export-odb-plus-plus_20250213_092028.jpg)

或者自定义导出的内容

![图 9](/storage/images/cn/pcb/export-odb-plus-plus/export-odb-plus-plus_20250213_092046.jpg)

在这里你可以自定义导出的单位，以及钻孔的相关内容。

![图 10](/storage/images/cn/pcb/export-odb-plus-plus/export-odb-plus-plus_20250213_092116.jpg)

下方可以增加不同的配置，在左侧点击即可对该配置进行修改或者导出
