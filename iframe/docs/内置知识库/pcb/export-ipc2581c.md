# 导出 IPC-2581C

嘉立创EDA支持导出 IPC-2581C 文件，该功能仅面向企业私有部署版本提供

**功能入口**:

- 顶部菜单栏 - 导出 - IPC-2581C

![图 0](/storage/images/cn/pcb/export-ipcd356a/export-ipcd356a_20250331_064334.jpg)

## 使用方法

点击 `导出 - 2581C` 后弹出`导出IPC-2581C文件`的弹窗.

![图 0](/storage/images/cn/pcb/export-ipc2581c/export-ipc2581c_20250331_064509.jpg)

在导出窗口中，支持编辑以下参数：

- 文件名称
- 单位（如毫米或英寸）
- OEM编号

![图 1](/storage/images/cn/pcb/export-ipc2581c/export-ipc2581c_20250331_064545.jpg)

完成参数配置后，点击“确认”按钮，系统将按照设置生成并导出 IPC-2581C 文件。
