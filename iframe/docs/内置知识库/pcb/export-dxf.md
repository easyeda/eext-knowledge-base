# 导出DXF

嘉立创EDA专业版支持强大的导出DXF功能。可以根据你需要创建多个配置，每个配置都可以自定义以下选项：选择要导出的具体图层；决定是否对图层进行镜像处理；调整图层的透明度；指定需要导出的对象。

**操作入口：**

-   顶部菜单 - 导出 - DXF
-   顶部菜单 - 文件 - 导出 - DXF。

![图 192](/storage/images/cn/pcb/export-dxf/export-dxf_20240621_172124.png)

在上方你可以选择AutoCAD版本
![图 0](/storage/images/cn/pcb/export-dxf/export-dxf_20240930_100625.png)

在左侧你可以预设一些配置，预设的配置导出一次后就会保存下来

![图 0](/storage/images/cn/pcb/export-dxf/export-dxf_20240621_172125.jpg)

右下方可以配置你的这个预设需要导出的层，这个层是否镜像，以及需要导出哪些对象

![图 1](/storage/images/cn/pcb/export-dxf/export-dxf_20240621_172126.jpg)

选择你预设的某一个配置，然后点击导出，即可导出一个对应的DXF文件

![图 3](/storage/images/cn/pcb/export-dxf/export-dxf_20240621_172127.jpg)

导出的DXF会根据你选择的层进行分层

![image-20201215194119735](/storage/images/en/pcb/export-dxf/export-dxf_20240621_172126.png)
