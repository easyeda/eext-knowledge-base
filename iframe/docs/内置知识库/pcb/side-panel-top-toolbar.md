# 顶部面板-工具栏

PCB顶面面板中的工具栏是一些常用的工具的一个快捷栏

![image-20210106133028092](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172123.png)

## 设置快捷栏

-   顶部菜单 - 设置 - 顶部工具栏

![image-20210520155317752](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172124.png)

在这弹窗可以设置原理图、符号、PCB、封装的顶部工具栏设置，图中红色框指的是各个界面的顶部菜单

![image-20210520155240381](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172125.png)

点击复选框可以查看对工具的分类显示

![image-20210520155338031](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172126.png)

## 自定义顶部工具栏

点击相应界面的顶部工具栏的设置

![image-20210520155350553](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172127.png)

弹窗的左边是可以添加到顶部工具栏的一些快捷工具

![image-20210520155404256](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172128.png)

弹窗的右边则是已经设置在顶部工具栏的快捷工具

![image-20210520155415735](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172129.png)

先在右侧选择选择需要添加快捷功能的位置，然后在左侧选择需要添加到左侧的快捷功能，点击中间的箭头或者把复选框打上勾，点击确定，即可添加到选择的位置上!

![image-20210520155436224](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172130.png)

![image-20210520155504048](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172131.png)

## 分类

在顶部工具栏中，每个功能快捷方式都会有一个分隔符作为分类的规划而这个分隔符则是可以在设置中进行修改。

![image-20210520155520902](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172132.png)

添加分隔符的方法与添加快捷功能的操作相似，先在右侧选中需要分割的地方，在从左侧选择分割符，点击中间的箭头，即可把分割符添加到顶部工具栏中，点击保存将当前设置保存并且应用到PCB界面上。

![image-20210520155536285](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172133.png)

可以看到，顶部工具栏的快捷功能被分隔开了

![image-20210520155556027](/storage/images/en/pcb/side-top-panel-toolbar/side-top-panel-toolbar_20240621_172134.png)
