# 3D外壳-顶面/底面挖槽区域

当需要绘制顶面和底面的挖槽时，使用顶底面挖槽区域。

**操作入口：**

-   顶部菜单 - 放置 - 3D外壳-顶面/底面挖槽区域

![图 23](/storage/images/cn/pcb/place-3d-shell-top-bottom-slot-region/place-3d-shell-top-bottom-slot-region_20240621_172124.png)

顶面/底面挖槽区域直接作用在顶层或者底层。

![图 22](/storage/images/cn/pcb/place-3d-shell-top-bottom-slot-region/place-3d-shell-top-bottom-slot-region_20240621_172125.png)

属性说明：

**图层**：设置在顶层外壳还是底层外壳。

**挖槽深度**：挖槽的深度。基于顶面或底面为起始面，进行下挖，或者上挖。自动校准会根据外壳的厚度，自动设置一个值确保可以挖穿。
