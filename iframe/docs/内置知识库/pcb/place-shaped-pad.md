# 异形焊盘

## 异形焊盘

异形焊盘，也称之为多边形焊盘，很多不规则形状的焊盘可以通过该功能进行绘制。比如一些天线也可以用异形焊盘进行绘制。

![图 34](/storage/images/cn/pcb/place-shaped-pad/place-shaped-pad_20240621_172124.png)

点击菜单进入绘制模式，和绘制填充区域类似，通过修改布线拐角（快捷键 L），可以切换不同的拐角，进而绘制不同的形状。

![图 35](/storage/images/en/pcb/place-shaped-pad/place-shaped-pad_20240621_172124.png)

点击右键完成绘制，设置焊盘的属性参数

![图 36](/storage/images/cn/pcb/place-shaped-pad/place-shaped-pad_20240621_172125.png)

确定完成绘制。

## 编辑形状

点击异形焊盘，在右边属性面板有一个按钮“编辑焊盘形状”。

![图 37](/storage/images/cn/pcb/place-shaped-pad/place-shaped-pad_20240621_172126.png)

点击后，光标进入编辑模式，可以拖动节点或双击节点进行编辑焊盘形状。

![图 38](/storage/images/cn/pcb/place-shaped-pad/place-shaped-pad_20240621_172127.png)
