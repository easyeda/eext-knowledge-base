## 添加网格设置

在网格设置弹窗中点击顶部的“+”号添加

![image-20210520161136462](/storage/images/en/pcb/view-grid-size/view-grid-size_20240621_172128.png)

双击刚刚添加的网格属性设置，修改完后点击确认，即可保存当前配置

![image-20210520161935362](/storage/images/en/pcb/view-grid-size/view-grid-size_20240621_172129.png)

## 删除网格设置

选择需要删除的网格大小设置，在网格设置弹窗中点击顶部的“X”号进行删除

![image-20210520161949509](/storage/images/en/pcb/view-grid-size/view-grid-size_20240621_172130.png)
