# PCB 对比

嘉立创EDA提供 PCB 对比功能，用于直观展示两个PCB文件之间的差异。

**功能入口**:

- 顶部菜单栏 - 工具 - PCB 对比...

![图 0](/storage/images/cn/pcb/tool-pcb-comparison/tool-pcb-comparison_20250331_081545.jpg)

## 使用方法

点击“PCB对比”后，在弹出的窗口中选择需要对比的两个PCB文件。支持选择当前工程内的PCB文件以及本地上传的PCB文件。

![图 1](/storage/images/cn/pcb/tool-pcb-comparison/tool-pcb-comparison_20250331_081702.jpg)

用户可自定义设置对比误差和对比尺寸的参数。

![图 2](/storage/images/cn/pcb/tool-pcb-comparison/tool-pcb-comparison_20250331_081744.jpg)

点击`开始对比`后，底部面板将显示`对比结果`标签。

![图 3](/storage/images/cn/pcb/tool-pcb-comparison/tool-pcb-comparison_20250331_081852.jpg)

中间表格将详细列出对比对象所在的图层及差异类型。

![图 4](/storage/images/cn/pcb/tool-pcb-comparison/tool-pcb-comparison_20250331_082010.jpg)

左侧树支持按对比对象所在图层和差异类型进行筛选。支持点击`PCB对比`按钮重新选择对比文件。

![图 5](/storage/images/cn/pcb/tool-pcb-comparison/tool-pcb-comparison_20250331_082129.jpg)

表格中的对比对象支持点击操作，点击后将在 PCB 中自动定位到对应的差异位置。

![图 6](/storage/images/cn/pcb/tool-pcb-comparison/tool-pcb-comparison_20250331_082331.gif)
