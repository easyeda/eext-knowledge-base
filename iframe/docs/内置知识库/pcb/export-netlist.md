# 导出网表

嘉立创EDA专业版目前只支持导出部分EDA工具的网表文件格式。

-   **顶部菜单 - 文件 - 导出 - 网表**

![图 0](/storage/images/cn/pcb/export-netlist/export-netlist_20240621_172124.jpg)

或者：**顶部菜单 - 导出 - 网表**

![图 1](/storage/images/cn/pcb/export-netlist/export-netlist_20240621_172125.jpg)

弹出窗口可以修改导出网表类型、需要导出网表的页面范围、相关参数和文件名。

![图 2](/storage/images/cn/pcb/export-netlist/export-netlist_20240621_172126.jpg)

-   网表类型：目前支持的网表类型有Allegro(.tel)、嘉立创EDA(专业版)(.enet)、PADS(.asc)、Protel2(.net)

-   范围：可以选择当前工程的任意一个原理图、PCB

-   附加属性-Value：网表类型为allegro时显示，为tel网表的一个可附加的属性，此处可以添加需要赋值的属性

-   附加属性-Tolerance：网表类型为allegro时显示，为tel网表的一个可附加的属性，此处可以添加需要赋值的属性

-   网表文件名：可以修改导出的网表文件名称
