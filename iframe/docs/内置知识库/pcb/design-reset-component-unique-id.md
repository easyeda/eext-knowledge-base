# 重置元件唯一 ID

嘉立创EDA专业版支持根据元件位号重置元件唯一 ID。

元件唯一 ID 用于确保原理图和 PCB 中元件的正确匹配，当出现元件不匹配的情况时，可以通过重置元件唯一 ID 来解决。

:::warning 注意
重置前请确保原理图和 PCB 的元件位号相对应。

重置后请把对应的原理图或者 PCB 也进行重置操作，原理图重置时将清空撤销/重做记录。
:::

**功能入口**：

顶部菜单栏 - 设计 - 重置元件唯一 ID

![图 0](/storage/images/cn/pcb/design-reset-component-unique-id/design-reset-component-unique-id_20241227_095844.jpg)

点击按钮后会出现警告弹窗，确保原理图和 PCB 的元件位号对应后继续操作。

![图 1](/storage/images/cn/pcb/design-reset-component-unique-id/design-reset-component-unique-id_20241227_095913.jpg)

操作后顶部出现信息提示，此时元件唯一 ID 重置。

![图 3](/storage/images/cn/pcb/design-reset-component-unique-id/design-reset-component-unique-id_20241227_100054.jpg)
