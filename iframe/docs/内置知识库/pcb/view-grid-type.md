# 网格类型

## 网格类型

支持切换两种网格类型：直角坐标系，极坐标系

![图 8](/storage/images/en/pcb/view-grid-type/view-grid-type_20240621_172123.png)

也可以设置画布内网格是格线还是格点，还是无网格显示。

## 直角坐标系

直角坐标系最常用，普通的板子均使用该坐标系即可完成设计。

![图 9](/storage/images/en/pcb/view-grid-type/view-grid-type_20240621_172124.png)

## 极坐标系

极坐标系是以圆心为原点，往外扇形外扩的一种坐标系，一般在设计环形板子和布局的时候使用。

![图 10](/storage/images/en/pcb/view-grid-type/view-grid-type_20240621_172125.png)
