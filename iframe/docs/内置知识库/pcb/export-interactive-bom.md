# 导出交互式BOM

支持导出交互式BOM文件，会把整个PCB 3D预览打包在一个html中，浏览器打开即可预览

入口：PCB - 顶部菜单 - 导出 - 交互式BOM

![picture 0](/storage/images/cn/pcb/export-interactive-bom/export-interactive-bom_20240621_172124.jpg)

你也可以在焊接辅助工具那里导出该交互式BOM文件。

入口：顶部菜单 - 工具 - 焊接辅助工具，导出 - 离线Html
