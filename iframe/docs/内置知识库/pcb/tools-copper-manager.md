# 铺铜管理器

铺铜管理器可以对铜皮批量管理。给铺铜增加优先级，修改网络、修改层、名称、铺铜的顺序改变。

**操作流程**：

-   **在顶部菜单 - 工具 - 铺铜管理器**

![image-20211224153957556](/storage/images/cn/pcb/tools-copper-manager/tools-copper-manager_20240621_172124.png)

![图 0](/storage/images/cn/pcb/tools-copper-manager/tools-copper-manager_20240621_172125.jpg)

铺铜管理器打开后，可以继续操作画布。在铺铜管理器中选中铺铜和画布中的选中将会保持同步，同时鼠标悬浮到铺铜管理器的某个铺铜后，画布中的对应铺铜将会高亮。

**上移、下移、置顶、置底、删除**

![图 1](/storage/images/cn/pcb/tools-copper-manager/tools-copper-manager_20240621_172126.jpg)

-   **上移**：在铺铜管理器中选中一个或多个铺铜后，通过点击上移将该铺铜优先级上移一位

-   **下移**：在铺铜管理器中选中一个或多个铺铜后，通过点击下移将该铺铜优先级下移一位

-   **置顶**：铺铜管理器中选中一个或多个铺铜后，通过点击置顶图标，将选中铺铜优先级置为这一层的最高

-   **置底**：铺铜管理器中选中一个或多个铺铜后，通过点击置底图标，将选中铺铜优先级置为这一层的最低

-   **删除**：铺铜管理器中选中一个或多个铺铜后，通过点击删除图标，将选中铺铜在列表中删除，这里的修改还未对画布生效，在点击应用或确认按钮后，画布中的铺铜区域将会被删除

**过滤**：可以在表格上方的输入框输入需要搜索的关键字进行铺铜的查找

![图 2](/storage/images/cn/pcb/tools-copper-manager/tools-copper-manager_20240621_172127.jpg)

**重建选中**：对当前选中的铺铜按照优先级顺序进行重建

**重建所有**：对所有铺铜分层并按照优先级顺序进行重建

注意：铺铜的顺序是根据铺铜的优先级进行，优先级越高，优先铺，铺铜结果先到先得。

**表格编辑**

-   **拖动排序**：每个铺铜前面有可以拖动排序的图标即表示该行可以进行拖动排序，长按拖动该行的前两列到其他行，可以快速将该铺铜的优先级进行调整

![图 3](/storage/images/cn/pcb/tools-copper-manager/tools-copper-manager_20240621_172128.jpg)

-   **修改优先级**：双击某个铺铜的优先级单元格，可以进行优先级的修改，修改后快速将该铺铜调整至对应优先级位置

![图 4](/storage/images/cn/pcb/tools-copper-manager/tools-copper-manager_20240621_172129.jpg)

-   **修改图层**：下拉可以在铺铜管理器中切换铺铜的图层，你也可以在右侧属性面板进行修改

![图 5](/storage/images/cn/pcb/tools-copper-manager/tools-copper-manager_20240621_172130.jpg)

-   **修改网络**：下拉可以在铺铜管理器中切换铺铜的网络，你也可以在右侧属性面板进行修改

![图 6](/storage/images/cn/pcb/tools-copper-manager/tools-copper-manager_20240621_172131.jpg)

**规则**：点击规则按钮可以快速打开设计规则的铺铜规则

**应用**：对以上修改进行保存但不关闭弹窗

**确认**：对以上修改进行保存同时关闭弹窗
