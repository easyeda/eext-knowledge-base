# V1.8.x 更新说明

## 原理图

-   支持查找网络标签

    ![图 0](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154226.png)

-   支持器件管理器，实现器件参数替换(不支持符号和封装同时替换)。

    **入口**：顶部菜单 - 工具 - 器件管理器

    ![图 1](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154256.png)

-   绘制导线总线时支持空格键切换布线方向
-   器件放置对话框和底部器件库增加申请绘制库按钮

    ![图 2](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154307.png)

-   查找全部对象时支持全部选中并高亮对象

-   移除导线拆分时的网络分割弹窗
-   属性值输入框支持希腊字符

    ![图 3](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154324.png)

-   普通文本输入框支持<kbd>ALT</kbd>/<kbd>SHIFT</kbd>+<kbd>ENTER</kbd>换行（网络名和位号等属性不支持）

    ![图 4](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154335.png)

-   支持导出PADS9.5的网表

    ![图 5](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154559.png)

-   放置网络端口和导线名称不一致时支持弹窗提示是否修改导线名

    ![图 6](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154608.png)

-   左侧树支持多选图页同时打开
-   底部面板新增设计规则Tab

    ![图 7](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154619.png)

-   导出BOM支持导出“加入BOM”和“转到PCB”属性，在左侧配置设置

    ![图 8](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154632.png)

-   原理图主题增加白底黑图和黑底白图

    ![图 9](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154638.png)

-   放置网络标识时支持TAB键设置全局网络名

    ![图 10](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154645.png)

-   移除网络端口时支持自动清除导线名称
-   不加入BOM和不转到PCB的元件不纳入设计规则检查
-   封装管理器的封装预览窗和封装列表右键增加编辑封装功能

    ![图 11](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154653.png)

-   删除工程库复用图块符号支持提示是否同时删除对应的原理图

    ![图 12](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154659.png)

-   <kbd>CTRL</kbd>+<kbd>ALT</kbd>移动导线名支持导线不跟随
-   重构导线布线逻辑，优化布线操作

## 面板

-   支持图元任意角度拖动旋转
-   面板3D预览支持设置背景颜色

    ![图 13](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154706.png)

-   多对象选中时，属性面板支持整体位置移动

    ![图 14](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_154712.png)

## PCB

-   支持复用组合的布局布线。使用方法：原理图绘制多个电路分组，更新到PCB，选中一组布局布线，选中已布局的对象，右键创建组合，其他模块也右键各自创建组合，选中第一个未完成布局布线的组合，**点击**：布局 - 复用布局布线，光标点选前面已完成布局布线的组合，会自动完成布局布线

-   支持焊盘对组的设置，方便进行等长布线。入口：顶部菜单 - 设计 - 焊盘对组管理器

-   PCB修改了外壳高度后支持同步到3D外壳预览
-   支持大量铺铜，大量自定义字体文本的PCB保存
-   支持预览版的自动布局(持续开发中)。入口：顶部菜单 - 布局 - 自动布局

    ![图 15](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155005.png)

-   封装编辑界面左侧tab支持封装向导

    ![图 16](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155030.png)

-   支持导出和导入自动布线文件

    ![图 17](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155037.png)

-   PCB一键下单支持先检测飞线，检测是否有边框，检测DRC
-   布线支持记录上次设置的线宽

-   3D外壳下单支持检测是否有3D外壳图元
-   调整元件选中高亮效果，移除高亮的盒子效果

    ![图 18](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155044.png)

-   缝合孔支持在所属网络的铺铜内添加
-   优化差分对布线的出线方式

-   支持元件右键编辑封装仅应用到当前元件

    ![图 19](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155051.png)

-   相对偏移功能支持记录上次的参数
-   支持框选元件进行拖动时，导线不断开
-   等长调节支持自动寻找等长起始位置
-   铺铜和内电层设计规则支持焊盘和插件焊盘的各自连接方式

    ![图 20](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155058.png)

-   下拉选项支持方向键选中

    ![图 21](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155104.png)

-   所有支持网络的对象均支持在属性面板手动新建网络
-   支持快捷键 <kbd>ctrl</kbd>+<kbd>SHIFT</kbd>+<kbd>V</kbd> 粘贴板子保持位号不变并隐藏飞线层，方便手动拼版
-   放置文本支持 <kbd>TAB</kbd> 键修改文本属性

    ![图 22](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155110.png)

-   等长调节对话框支持设置需要新增的长度。在等长调节时按 <kbd>TAB</kbd> 键弹出。

    ![图 23](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155124.png)

-   左侧网络 <kbd>Tab</kbd> 支持对网络类，差分对，焊盘对等进行导线长度排序

    ![图 24](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155134.png)

-   支持绘制自相交的多变形图元，绘制后自动拆分多个不相交的图元
-   指定间距分布支持以中心间距和边沿间距两种方式

    ![图 25](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155140.png)

-   等长网络组支持区分出已经归属其他网络组的网络

    ![图 26](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155146.png)

-   3D预览增加导入变更按钮，支持导入 PCB 的变更

    ![图 27](/storage/images/cn/release-note/v1.8.x/v1.8.x_20241126_155155.png)

-   优化焊盘出线和入线的扇出
-   差分对布线起始布线和打孔时支持按空格键切换布线拐角
-   拉伸导线融合节点后支持继续拉伸
-   圆弧模式布线结束时支持以圆弧模式完成
-   元件切层快捷键默认留空
-   支持封装内相同焊盘编号的焊盘设置不同的网络名
