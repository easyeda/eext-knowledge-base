## PCB

- 全新GPU显示引擎，移除旧WebGL引擎，使用WebGPU提供更强性能，流畅支持10万焊盘平移与缩放，需要使用最新版本的谷歌或者Edge浏览器，火狐浏览器暂未支持WebGPU
- 支持CPU显示引擎，在显卡支持不佳情况下自动切换至CPU显示引擎
- 全新的高性能铺铜算法，支持30万焊盘8层LED灯板45秒完成(某传统友商约5分钟)
