# V1.5.x 更新说明

## 原理图

-   支持放置网络标签在引脚上自动生成导线

![图 0](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114313.png)

-   导线绘制未确定线调整为半透明以做区分

![图 1](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114501.png)

-   放置器件支持选择是否更新掉当前工程库的旧版本器件模板

![图 2](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114506.png)

-   放置端口支持TAB设置名称

![图 3](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114512.png)

-   支持框选部分电路整体旋转
-   支持左侧栏基础库，支持自定义基础库

![图 4](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114518.png)

-   端口修改名称时不自动显示导线的名称
-   支持检测当前画布的元件数，支持在原理图设置检测数量

![图 5](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114526.png)

-   支持直接删除导线的名称，之前是隐藏逻辑
-   支持放置器件，旋转器件的时候位号等属性的自动调整摆放位置
-   支持导线右键高亮网络，快捷键 H；在左侧网络树点击支持临时高亮导线

![图 6](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114559.png)

-   支持鼠标悬浮导线导线高亮同网络导线。在视图菜单 - 高亮网络开启

![图 7](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114611.png)

-   放置网络标签支持快捷键 N
-   支持点击画布的属性值，按 TAB 键循环选中父级图元
-   支持显示图页名称在图纸属性
-   导出BOM支持选择导出实时的价格和库存

![图 8](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114617.png)

-   底部元件库支持显示实时的价格和库存

![图 9](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114622.png)

-   支持导线在布线结束后自动移除回路

## PCB

-   支持外壳绘制(上下壳和推盖两种类型)。入口：放置 - 3D外壳图元；支持导出3D外壳文件。

![图 10](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114710.png)

![图 11](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114715.png)

![图 12](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114720.png)

-   3D外壳预览支持独立弹窗

    **入口**：视图 - 3D外壳预览

![图 13](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_114728.png)

-   2D 和 3D 预览支持导出图片，3D 预览支持导出 3D 文件

![图 14](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115015.png)

-   支持拖动多边形区域的边界线段调整大小

![图 15](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115019.png)

-   3D模型管理器支持按百分比调整模型缩放比例

![图 16](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115024.png)

-   布线结束后支持自动扇出
-   支持移动元件属性文本时高亮元件

![图 17](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115031.png)

-   禁止区域的禁止选项拆分多个类型

![图 18](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115041.png)

-   拼板外框支持圆角倒角

![图 19](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115050.png)

-   DRC错误标识样式调整，放大PCB不会放大DRC标识

![图 20](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115056.png)

-   移动铺铜的时候支持自动移除铺铜填充，支持撤销移动
-   新的差分对和网络类设置窗口，增加网络类设置入口

![图 21](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115105.png)

![图 22](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115118.png)

![图 23](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115121.png)

-   普通过孔类型点击切换到盲埋孔类型支持弹窗设置盲埋孔规则

![图 24](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115126.png)

-   默认文本字体切到反相设置支持弹窗提示切换到其他字体

![图 25](/storage/images/cn/release-note/v1.5.x/v1.5.x_20241127_115133.png)
