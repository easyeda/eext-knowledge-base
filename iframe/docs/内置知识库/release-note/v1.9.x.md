# V1.9.x 更新说明

::: warning 注意

1. 这个版本的快捷键新增了一个“嘉立创EDA专业版”主题，很多功能的快捷键有修改，请务必注意新的快捷键
2. 这个版本支持在设置里面隐藏符号库的单独入口，确保建库交互和标准版基本保持一致

:::

## 原理图

-   符号库管理器支持简易模式和专业模式切换。

    **入口**：设置 - 系统 - 通用 - 符号库管理。简易模式将隐藏单独的符号编辑入口，合并并简化符号和器件的创建编辑流程

![图 0](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144238.png)

-   支持设置原理图导线的默认网络名规则，支持位号+引脚编号成为网络名。

    **入口**：设置 - 原理图 - 通用 - 默认网络名

![图 1](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144256.png)

![图 2](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144313.png)

-   符号库支持规格书提取向导。

    **入口**：符号库 - 工具 - 规格书提取向导。

    **使用方法**：打开规格书元件引脚定义页，截图，粘贴在规格书提取向导对话框，会进行OCR识别出引脚名和引脚编号，下一步创建符号库即可。

![图 3](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144331.png)

![图 4](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144340.png)

-   支持复制粘贴没有显示网络名称的导线时，自动清除导线名称
-   支持多部件元件的部件切换。

    **入口**：多部件元件 - 属性面板 - 部件名称属性点击切换

![图 5](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144358.png)

-   删除器件库的时候，支持选择是否同时删除关联的符号和封装

![图 6](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144411.png)

-   支持在底部库放置图纸器件

![图 8](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144424.png)

-   支持更多的方向顺序分配位号

![图 9](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144437.png)

-   优化方向键移动图元
-   优化器件管理器的器件替换逻辑。支持替换符号和封装和3D模型

![图 10](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144445.png)

-   放置文本支持设置更多样式

![图 11](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144455.png)

-   原理图和符号库支持导入DXF

![图 12](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144503.png)

-   支持器件标准化设置。

    **入口**：左侧器件标准化Tab。支持把没有分配立创编号的器件进行分配

![图 13](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144521.png)

-   元件右键菜单支持导线扇出。

    **入口**：元件右键菜单 - 扇出网络标签/非连接标识

![图 14](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144529.png)

-   图纸右侧属性面板支持直接调整图纸大小，支持切换图纸尺寸

![图 15](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144543.png)

-   搜索C9900开头的商品编号器件时支持自动切换到嘉立创EDA引擎
-   元件右边属性面板增加显示“关键属性”和“更多属性”分组折叠

## PCB

-   等长布线时支持更小的高度调整
-   支持差分对等长布线。
    入口：布线 - 差分对等长布线

![图 18](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144641.png)

-   优化Gerber的焊盘表示方式，使板厂可以正常识别焊盘
-   支持空网络图元的过滤和拾取，左侧树支持查看空网络

![图 19](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144657.png)

-   3D轮廓视图调整为只对3D外壳进行轮廓查看
-   支持根据当前鼠标位置平移画布到中央，快捷键 SHIFT+HOME
-   支持数字键3，4进行切换内层3，内层4
-   支持放置或粘贴封装时询问是否开启当前PCB不支持的层
-   支持尺寸图元画在用户自定义层
-   优化取消组合，支持单个对象移出组合
-   选中泪滴支持属性面板，支持手动删除泪滴

![图 20](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144707.png)

-   删除元件支持弹窗提示，不再提示只对当前PCB有效，重新打开PCB会再次提示

![图 21](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144714.png)

-   优化布线违反DRC时的添加过孔的处理
-   导出DXF增加3D外壳相关图元的导出
-   选择导线后，属性面板增加网络长度

![图 22](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144721.png)

-   绘制铺铜、填充区域、线条支持继承最先接触到的图元的网络
-   优化实时DRC，在图元绘制后再检测，避免绘制过程中的卡顿
-   内电层区块支持设置是否保留孤岛

![图 23](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144739.png)

-   支持自定义部分设计规则进行检查。

    **入口**：设计 - 检查DRC(自定义)

![图 24](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144750.png)

![图 25](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144755.png)

-   支持从PCB更新到原理图。

    **入口**：设计 - 更新PCB到原理图。将根据元件唯一ID把PCB的元件属性更新到原理图

![图 26](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144814.png)

-   画布右边属性面板的网格和栅格设置支持等比例联动

![图 27](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144834.png)

-   画布右边属性面板的设置支持存入文件中
-   支持PCB重新分配位号。

**入口**：设计 - 分配位号

![图 28](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144839.png)

-   导线支持在空白区域进行绘制
-   支持复用图块工程创建PCB，
-   支持原理图转PCB时，调用复用图块工程的PCB。

    **入口**：新建 - 复用图块，会自动创建PCB。

    已调用的PCB模块支持再次从复用图块工程的PCB更新布局入口。设计 - 从复用模板更新布局布线

![图 29](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144849.png)

-   PCB内置默认字体支持Ω和μ字符

![图 30](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144857.png)

-   只选中元件的属性时，元件也支持被组合
-   复用布局布线功能支持复用元件的丝印文本等位置
-   支持放置封装时默认字体和文本大小设置。

    **入口**：设置 - PCB - 通用 - 元件属性默认字体。

    如果封装编辑时已经设置属性的文本大小则优先使用编辑时的大小和字体。

![图 31](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144903.png)

**入口**：画布右边属性面板 - 拉伸导线最小拐角

![图 32](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144911.png)

![图 33](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144919.png)

-   拖动封装时，支持按镜像的快捷键对封装进行整体镜像

-   支持焊盘连接功能，方便无原理图PCB快速生成网络。

**入口**：工具 - 焊盘连接

![图 34](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144928.png)

-   设计规则焊盘的阻焊扩展默认改为 2mil
-   铺铜属性面板的移除缝合孔支持根据缝合孔尺寸移除

![图 35](/storage/images/cn/release-note/v1.9.x/v1.9.x_20241126_144954.png)

-   导出的Gerber新增一个Drill_PTH_Through_Via.DRL文件来区分通孔型过孔的钻孔文件
-   在非信号层布线支持自动切换到顶层进行布线
