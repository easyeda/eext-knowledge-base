# V1.7.x 更新说明

## 原理图

-   检查DRC支持输出未分配位号，重复位号的提示日志，增加分配位号规则和位号重复检测规则。

    **入口**：设计 - 设计规则

    ![图 0](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241126_164219.png)

-   支持批量拖动过程中旋转翻转
-   优化符号库放置引脚时的引脚名称太近的问题

    ![图 1](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241126_164356.png)

-   增加网络标签(网络名)拖动类型切换设置：调整属性位置(默认)；修改网络名。修改网络名类型和标准版的网络标签交互类似。

    **入口**：设置 - 原理图 - 常用

    ![图 2](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241126_164407.png)

    ![图 3](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241126_164424.png)

-   勾选旋转元件自动属性调整位置时，待放置的元件旋转时支持同时旋转属性位置
-   画布右侧属性面板支持显示图纸大小，支持编辑图标，点击可直接编辑图纸模板；设计菜单的工程图纸预设弹窗支持编辑按钮

    ![图 4](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241126_164437.png)

    ![图 5](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241126_164446.png)

-   创建原理图默认图纸大小调整为A4
-   修改导线的网络名时，支持弹窗提示是否修改连接的网络标识名称

    ![图 6](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241126_164459.png)

-   放置器件时支持按 <kbd>Tab</kbd> 设置名称和位号

    ![图 7](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241126_164510.png)

-   主题设置增加对引脚和属性和网络标签等设置

    ![图 8](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_104923.png)

-   导线合并提示弹窗增加放置短接符按钮

    ![图 9](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_105020.png)

-   移除放置器件弹窗在结束放置后再次弹窗的行为

## PCB

-   支持PCB内直接修改封装的焊盘尺寸和焊盘的坐标位置

![图 10](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_105035.png)

-   支持拖动过孔时，联动平移导线
-   3D外壳挖槽区域属性面板增加挖槽选项

![图 11](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_105044.png)

-   支持在3D外壳设计时，从顶面或底层内部往外部挖槽
-   在PCB设置里增加一个选项：使用嘉立创彩色丝印工艺。默认关闭，开启后图元属性面板展示对应彩色丝印的属性。

    **入口**：设置 - PCB - 常用

    ![图 12](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_105745.png)

-   支持在3D外壳增加下壳内壁厚度属性

![图 13](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_105802.png)

-   支持在3D外壳增加螺丝柱属性设置，支持加强筋设置，支持沉头孔设置

![图 14](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_105811.png)

-   一键导出Gerber的单位和格式设置移入自定义设置里面

![图 15](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_105850.png)

-   支持绘制线条结束后自动合并同一条直线上的线段
-   矩形图元支持在属性面板设置圆角半径属性

![图 16](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_105855.png)

-   顶部工具栏支持放置焊盘和对齐功能。需要手动恢复默认添加。

    **入口**：设置 - 顶部工具栏 - PCB

![图 17](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_105905.png)

-   支持在3D外壳设计时，超出壳体的螺丝柱自动裁剪掉
-   拆分网格和栅格设置。

    **入口**：画布 - 右侧属性面板

![图 19](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110108.png)

-   删除板框时支持自动删除内电层区块
-   过滤选择焊盘对或网络时，拖动导线仅调整单段导线，不再整体被拖动
-   轮廓对象支持右键打散为独立线条

![图 20](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110115.png)

-   PCB设置增加“自动结束布线”选项，不勾选时，导线等待手动结束不再自动结束。

    **入口**：设置 - PCB - 常用

![图 21](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110125.png)

-   高亮和取消高亮快捷键改为同一个 <kbd>H</kbd>，飞线隐藏显示快捷键<kbd>CTRL</kbd>+<kbd>R</kbd>改为循环显示隐藏，增加循环显示隐藏网络名快捷键<kbd>CTRL</kbd>+<kbd>Q</kbd>(该快捷键临时生效)
-   布线进入焊盘被阻挡时支持自动翻转路径
-   放置菜单新增3D外壳实体图元

![图 22](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110131.png)

-   3D外壳设计图元菜单添加图标
-   支持设置网络颜色。

**入口**：左侧面板 - 网络Tab - 网络或网络类。

点击网络名前的颜色设置方块设置。

![图 23](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110225.png)

-   导出的坐标文件Device列改为Comment列，以兼容部分贴片机
-   PCB吸附增加快捷键 <kbd>ALT</kbd> + <kbd>S</kbd>

![图 24](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110246.png)

-   优化过滤Tab，勾选全部不再勾选其他分类的选项
-   布线时检测到违反DRC间距时，支持在对应图元外面显示白色轮廓以作提示

![图 25](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110256.png)

-   优化阵列功能，支持设置图元的旋转角度，调整旋转对象以适合圆形的逻辑和标准版一致
-   支持翻面功能，对所选的对象(需包含元件)进行整体翻面，镜像并且换层。

    **入口**：布局 - 翻面

![图 26](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110307.png)

-   设计菜单增加“清除错误”菜单

![图 27](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110315.png)

-   设计菜单点击“实时DRC”菜单时，弹窗选择是否立即检测DRC

![图 28](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110322.png)

-   导入DXF支持自动融合重复的线条
-   差分对绘制支持快捷键 <kbd>ALT</kbd>+<kbd>D</kbd>，等长调整支持快捷键 <kbd>SHIFT</kbd>+<kbd>A</kbd>

![图 29](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110328.png)

-   PCB的封装焊盘右键支持移动菜单

![图 30](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110333.png)

-   圆弧类型的导线段属性支持切换圆弧类型：中心圆弧或两点圆弧

![图 31](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110339.png)

-   DRC错误列表支持点击规则名直接打开规则设置对话框

![图 32](/storage/images/cn/release-note/v1.7.x/v1.7.x_20241127_110343.png)
