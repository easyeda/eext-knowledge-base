# 其他常见问题

## 嘉立创EDA专业版推荐配置 {#配置}

使用嘉立创EDA专业版时，可以参加以下设备的推荐配置。

提示：嘉立创EDA专业版在V2.2开始支持CPU和WebGPU双引擎，当电脑没有显卡或显卡支持不佳时，会自动切换到CPU引擎，后续可能会提供手动切换方式。CPU引擎比WebGPU引擎会慢一些，内存占用也会相应大一些。

**嘉立创EDA专业版用户配置**：

| 设备   | 描述                                                                                                                                                               |
| :----- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| CPU    | 2 核心及以上，推荐 i3-9100 及以上配置，支持国产 CPU 架构 X86、AMD64、LoongArch64、SW_64、ARM64                                                                     |
| 内存   | 1 万 Pin 以内 8G DDR4 内存及以上，1 万 Pin 以上推荐 16GB 及以上                                                                                                    |
| 显卡   | 2017 年后的独立显卡，推荐 GTX1080 及以上，X86 CPU 时至少是 2019 年后的集显。显卡驱动升级到最新。国产 CPU 时需要独立显卡                                            |
| 磁盘   | 系统盘至少 128G SSD，数据盘至少 100G HDD，推荐 SSD                                                                                                                 |
| 浏览器 | Chrome、Edge、Firefox 最新版，推荐 Chrome。至少 V120 版及以上，64 位版本，并开启硬件加速。国产浏览器内核版本太低不支持。                                           |
| 系统   | 1. Windows：Windows 10 及以上系统，不兼容 Windows 7 和 Windows XP。2. Linux：Linux 内核 4.9 及以上，支持 Ubuntu、UOS、deepin、Kylin 等。3. Mac：MacOS 10.12 及以上 |

**嘉立创EDA专业版私有部署配置**：

| 设备     | 描述                                                                                                             |
| :------- | :--------------------------------------------------------------------------------------------------------------- |
| 使用人数 | 100人内                                                                                                          |
| CPU      | Intel 处理器 2 核心及以上，推荐 i3-9100 及以上配置                                                               |
| 架构     | X86、AMD64                                                                                                       |
| 内存     | 16G DDR4-2666 内存及以上                                                                                         |
| 磁盘     | 系统盘至少 128G SSD，数据盘至少 100G SSD，HDD 亦可但可能部分功能性能表现不理想                                   |
| 系统     | 1. Windows：Windows Server 2016 或 Windows 10 及以上系统。2. Linux：Linux 内核 4.9 及以上，支持 CentOS/Ubuntu 等 |
