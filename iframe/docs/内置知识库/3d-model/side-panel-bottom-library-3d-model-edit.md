# 编辑3D库

嘉立创EDA专业版支持替换3D库的3D模型。

在底部元件库，找到3D模型后，点击个人的3D模型列表，右键进行替换3D模型

![图 1](/storage/images/cn/3d-model/side-panel-bottom-library-3d-model-edit/side-panel-bottom-library-3d-model-edit_20240621_172123.png)

也可以右键修改3D模型标题等信息。

![图 2](/storage/images/cn/3d-model/side-panel-bottom-library-3d-model-edit/side-panel-bottom-library-3d-model-edit_20240621_172124.png)
