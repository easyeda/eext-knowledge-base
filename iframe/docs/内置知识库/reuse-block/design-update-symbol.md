# 更新符号

嘉立创EDA专业版支持批量修改更新符号、封装和复用模块。

在修改错误完的器件符号保存后，可不用再重新放置一个，嘉立创EDA专业版支持原理图更新符号。

**更新符号方法**：

-   顶部菜单 - 设计 - 更新器件

-   选中器件 - 鼠标右键- 更新器件

![image-20211227135159874](/storage/images/en/reuse-block/design-update-symbol/design-update-symbol_20240621_172123.png)

点击选择，即可显示最新修改的符号，点击确定即可应用到原理图中。更新封装操作与更新符号操作一致，这里就不在再讲解了
