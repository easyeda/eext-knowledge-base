# IPCB\_PrimitiveLine class

直线图元

## 签名

```typescript
declare class IPCB_PrimitiveLine implements IPCB_Primitive
```
**实现自：**[IPCB\_Primitive](./pro-api.ipcb_primitive.md)

## 备注

直线和圆弧线均为导线，对应画布的线条走线和圆弧走线
