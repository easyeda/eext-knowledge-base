# PCB如何快速选择操作

&emsp;&emsp;在PCB里，使用“编辑-选择对象”功能，可以快捷选中所需要的元器件，提高设计效率。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-select-object/pcb-select-object_20251128_135339.png"  style="border:2px solid gray" width="350"></center>

&emsp;&emsp;①单击鼠标左键，进行单个选择。

&emsp;&emsp;②按住“Ctrl”件，多次点击鼠标左键，进行多个选择。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-select-object/pcb-select-object_20251128_141511.png"  style="border:2px solid gray" width="800"></center>

&emsp;&emsp;③按照鼠标左键，从左往右框选，框选范围里面的对象都会被选择到，框选外面的或框选搭边的对象，将无法被选中。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-select-object/pcb-select-object_20251128_141812.png"  style="border:2px solid gray" width="500"></center>

&emsp;&emsp;④按照鼠标左键，从右往左框选，框选范围里面以及和框选搭边的对象都会被选择到，框选外面的对象，就还是无法被选中。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-select-object/pcb-select-object_20251128_142245.png"  style="border:2px solid gray" width="500"></center>

&emsp;&emsp;⑤全选（快捷键：Ctrl+A），PCB的全部对象都会被选中。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-select-object/pcb-select-object_20251128_142435.png"  style="border:2px solid gray" width="800"></center>

&emsp;&emsp;⑥选择对象-矩形内部（快捷键：ESI），框选范围里面的对象会被选中，框选完后可再框选，完成选择后，按ECS键退出此模式。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-select-object/pcb-select-object_20251128_142712.png"  style="border:2px solid gray" width="500"></center>

&emsp;&emsp;⑦选择对象-矩形外部（快捷键：ESO），框选范围里面的对象不会被选中，框选外的其他对象将被选中，即反选功能，框选完成选择后，按ECS键退出此模式。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-select-object/pcb-select-object_20251128_143600.png"  style="border:2px solid gray" width="800"></center>

&emsp;&emsp;⑧选择对象-多边形内部，与“选择对象-矩形内部“功能类似，用光标绘制的多边形进行选择，在多边形区域里面的对象将被选中。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-select-object/pcb-select-object_20251128_144307.png"  style="border:2px solid gray" width="500"></center>

&emsp;&emsp;⑨选择对象-多边形外部，与“选择对象-矩形外部“功能类似，用光标绘制的多边形进行选择，在多边形区域里面的对象不会被选中，区域外的对象将被选中。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-select-object/pcb-select-object_20251128_145045.png"  style="border:2px solid gray" width="800"></center>

&emsp;&emsp;⑩选择对象-接触到线条的（快捷键：ESL），用光标绘制线条进行选择，线条接触到的对象将被选中。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-select-object/pcb-select-object_20251128_150103.png"  style="border:2px solid gray" width="800"></center>

&emsp;&emsp;⑪选择对象-连接的铺铜（快捷键：ESP或Ctrl+H），鼠标左键点击选择的对象，对象所连接的网络铺铜将会被选中。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-select-object/pcb-select-object_20251128_150737.png"  style="border:2px solid gray" width="800"></center>

&emsp;&emsp;⑫交叉选择（快捷键：Shift+X），在同一板子下，点击选择中原理图或PCB的元件后，使用设计-交叉选择，原理图和PCB中的这个元件会相互跳转高亮显示定位，快速找到其位置，无需来回查看。
