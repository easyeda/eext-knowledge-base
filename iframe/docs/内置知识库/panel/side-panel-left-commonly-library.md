# 左侧面板-常用库

## 左侧面板-常用库

-   左侧面板 - 常用库 - 面板

![图 0](/storage/images/cn/panel/side-panel-left-commonly-library/side-panel-left-commonly-library_20240621_172124.png)

当前编辑页为面板时，可通过左侧的常用库面板选择面板库，快捷的进行面板库放置。

![图 1](/storage/images/cn/panel/side-panel-left-commonly-library/side-panel-left-commonly-library_20240621_172125.png)

**过滤查找**

支持输入过滤条件进行特定的查找。

![图 2](/storage/images/cn/panel/side-panel-left-commonly-library/side-panel-left-commonly-library_20240621_172126.png)

**个人库&系统库**

个人库需要自己选择面板库添加到常用库中。
![图 3](/storage/images/cn/panel/side-panel-left-commonly-library/side-panel-left-commonly-library_20240621_172127.png)

系统库是官方提供的常用示例面板库。
![图 4](/storage/images/cn/panel/side-panel-left-commonly-library/side-panel-left-commonly-library_20240621_172128.png)

### 添加常用库

单击左侧常用库面板左上的设置图标，弹出的设置弹窗，可进行常用库的自定义。

![图 5](/storage/images/cn/panel/side-panel-left-commonly-library/side-panel-left-commonly-library_20240621_172129.png)

**常用库分类**

单击分类列表中的绿色加号图标可进行常用库分类的添加，支持自定义分类名。

![图 7](/storage/images/cn/panel/side-panel-left-commonly-library/side-panel-left-commonly-library_20240621_172130.png)

**常用库添加**

单击右侧面板库信息列表上方的绿色加号图标，会弹出选择面板库的弹窗，可自定义添加常用库。

![图 18](/storage/images/cn/panel/side-panel-left-commonly-library/side-panel-left-commonly-library_20240621_172131.png)

还可以在底部面板库中选中面板库，右键菜单操作添加到常用库

![图 8](/storage/images/cn/panel/side-panel-left-commonly-library/side-panel-left-commonly-library_20240621_172132.png)
