# 分布

## 分布

选择的多个图元进行等间距分布排列，等距排列分布的前提是要在选择两个图元以上才能进行排列分布。
等距分布过程中图元不吸附格点，即不受当前网格尺寸影响，会保证每相邻的两个图元之间的间距值都是一致的。

### 水平等距分布

在水平方向上将所有选中的图元进行等距分布排列。

**操作步骤**：

选择需要排列分布的图元后

-   顶部菜单 - 布局 - 分布 - 水平等距分布

-   全局快捷键：<kbd>Ctrl</kbd> + <kbd>Shift</kbd>+ <kbd>H</kbd>

### 垂直等距分布

在垂直方向上将所有选中的图元进行等距分布排列。

**操作步骤**：

选择需要排列分布的图元后

-   顶部菜单 - 布局 - 分布 - 垂直等距分布

-   全局快捷键：<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd>
