# 放置SVG

## 放置SVG

嘉立创EDA专业版支持导入SVG做为路径进行编辑，⽬前还没有做到1:1，导⼊后注意查看尺⼨，导⼊SVG-如果选择导⼊为形状，则可以对颜⾊等信息进⾏修改，选择导⼊为图⽚则不可更改。 适⽤于⼀些LOGO和图标。

### 操作步骤

-   顶部菜单 - 放置 - SVG

![image-20220729155419605](/storage/images/cn/panel/place-svg/place-svg_20240621_172123.png)

导入的时候会弹窗提示选择导入类型：

![image-20220729155440561](/storage/images/cn/panel/place-svg/place-svg_20240621_172124.png)

选择导入形状后，可以直接编辑其形状。

![image-20220729155715920](/storage/images/cn/panel/place-svg/place-svg_20240621_172125.png)
