# 吸附

## 吸附

在绘制图元的时候，支持显示吸附十字小标识和红色的对齐指示线，移动图元时，出现吸附标识的位置都可以把光标或图元吸附点吸附过去。

吸附默认开启，暂不支持关闭吸附。

![图 102](/storage/images/en/panel/edit-snap/edit-snap_20240621_172122.png)

栅格吸附距离可以顶部工具栏修改网格尺寸。

![图 0](/storage/images/cn/panel/edit-snap/edit-snap_20240621_172123.png)

**设置** - **面板/面板库** - **通用**，ALT栅格尺寸也可以调整默认网格尺寸。

![图 1](/storage/images/cn/panel/edit-snap/edit-snap_20240621_172124.png)
