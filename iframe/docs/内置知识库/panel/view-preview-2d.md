# 2D预览

## 2D预览

面板设计支持2D预览查看效果，可避免一些设计上的错误，也能看到面板最终的设计样图。

### 功能入口

-   顶部菜单 - 视图 - 2D预览

![图 1](/storage/images/cn/panel/view-preview-2d/view-preview-2d_20240621_172124.png)

-   直接单击顶部工具栏中的2D预览图标，也可快速打开2D预览。

![图 2](/storage/images/cn/panel/view-preview-2d/view-preview-2d_20240621_172125.png)

**2D预览弹窗**

为了确保预览效果能够接近实际生产的水准，我们需要进行图形的精确计算。图形的复杂性越高，所需的计算和渲染时间就越长，从而导致加载过程需要更多的时间来完成。

![图 3](/storage/images/cn/panel/view-preview-2d/view-preview-2d_20240621_172126.png)

点击右侧眼睛，可以控制层的显示隐藏。
