# 右侧面板 - 过滤

## 右侧面板 - 过滤

在右侧过滤面板中可以很方便进行过滤所需要的对象。
![图 10](/storage/images/cn/panel/side-panel-right-filter/side-panel-right-filter_20240621_172124.png)

### 过滤配置

右侧过滤面板中，顶部下拉框可以切换过滤配置。

![图 11](/storage/images/cn/panel/side-panel-right-filter/side-panel-right-filter_20240621_172125.png)

修改配置后可以操作另存为配置，弹出自定义配置名称的输入弹窗，完成输入后单击确认即可完成配置的新建保存，方便后续快速切换和使用。

![图 12](/storage/images/cn/panel/side-panel-right-filter/side-panel-right-filter_20240621_172126.png)

还可以设置自定义的配置为默认。

![图 13](/storage/images/cn/panel/side-panel-right-filter/side-panel-right-filter_20240621_172127.png)

自定义的配置还支持重命名、保存配置、删除配置。

![图 14](/storage/images/cn/panel/side-panel-right-filter/side-panel-right-filter_20240621_172128.png)

同时打开多个面板，可以设置不同的过滤配置。便于针对不同的面板进行不同的过滤筛选。

新建的面板过滤配置都会是"常用(默认)"。
新打开的面板则会保持其原本的默认过滤配置。
刷新浏览器则会保持刷新前各面板原本的过滤配置。
删除当前设置的过滤配置，会重置为常用配置，常用配置是无法删除的。

### 显示/隐藏

可设置在面板画布中显示或隐藏某类型的元素。

![图 3](/storage/images/cn/panel/side-panel-right-filter/side-panel-right-filter_20240621_172129.png)

点击旁边的小眼睛，将在画布上批量隐藏对应类型的元素。

### 勾选项

设定筛选条件可以通过单击选项标签来选择或取消对应的选项。取消过滤类型名称前面的勾选后则在画布上无法单击或框选选中，勾选时则可在画布上选中。

![图 5](/storage/images/cn/panel/side-panel-right-filter/side-panel-right-filter_20240621_172130.png)

默认锁定类型的对象是过滤的。

![图 2](/storage/images/cn/panel/side-panel-right-filter/side-panel-right-filter_20240621_172131.png)

### 过滤类型

图元生效与否和锁定与否的相关属性设置在每个图元的右侧属性面板中

-   **状态**：根据元素的状态属性类型进行过滤。

-   **元素类型**：按照面板的元素类型进行过滤

![图 6](/storage/images/cn/panel/side-panel-right-filter/side-panel-right-filter_20240621_172132.png)
