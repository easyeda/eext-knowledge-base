# 面板下单

嘉立创EDA专业版面板编辑器支持一键下单和导出面板文件下单，如使用的是一键下单的流程，那就无需再次导出上传文件到立创商城。

**注意：**

-   导出的下单文件.epanm手动上传至立创商城下单需要文件大小在8M以下。一键下单无需顾虑此问题，建议使用一键下单功能。

## 一键下单

**下单步骤**：

-   顶部菜单栏 - 下单 - 面板下单

![图 2](/storage/images/cn/panel/order-order-panel/order-order-panel_20240621_172123.png)

预览没有问题后，点确认按钮。

![图 0](/storage/images/cn/panel/order-order-panel/order-order-panel_20240621_172124.png)

待下单数据生成完成后点击确定即可跳转至立创商城面板的下单页面，将工艺选择好后即可下单。

![图 98](/storage/images/cn/panel/order-order-panel/order-order-panel_20240621_172125.png)

## 导出面板制造文件

导出面板制造文件的格式是为.epanm，属于嘉立创EDA专业版的专用格式，只能用于立创商城面板下单使用，不能到其他的网站使用。

通过：顶部菜单 - 导出 - 面板制造文件

![图 95](/storage/images/cn/panel/order-order-panel/order-order-panel_20240621_172126.png)

![image-20220728153859258](/storage/images/cn/panel/order-order-panel/order-order-panel_20240621_172127.png)

预览没有问题后，点导出按钮。

![图 1](/storage/images/cn/panel/order-order-panel/order-order-panel_20240621_172128.png)

等待数据生成后选择导出即可把下单文件导出个人用户至本地中，然后可以前往立创商城下单：[面板打印下单](https://dos.szlcsc.com/dos/panel/print.html)
