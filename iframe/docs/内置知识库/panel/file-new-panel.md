# 新建面板

## 新建操作

在线打开地址：http://pro.lceda.cn/editor#，进⼊专业版才能进⾏⾯板图形绘制。 在线访问嘉⽴创EDA专业版，也推出了客户端。客户端下载链接：https://lceda.cn/page/download

**新建注意说明**：面板不能单独新建，需要新建一个工程后才能新建面板设计图。

1、⾸先在编辑区新建⼯程，填写⼯程名字，⽤于管理⾯板⽂件

-   顶部菜单 - 新建 - 工程
    ![1](/storage/images/cn/panel/file-new-panel/file-new-panel_20240621_172123.png)

2、新建完成工程后再新建面板

-   顶部菜单 - 新建 - 面板
    ![图 48](/storage/images/cn/panel/file-new-panel/file-new-panel_20240621_172124.png)

## 设计图说明

-   **定位点**：设计图纸上的红丝边框四个角（1、2、3、4点）中都有一个四方形的框，那个是板材的定位点，用于生产时候定位使用，默认不能修改。
-   **板材框**：设计图中红色的一个矩形框是立创商城面板打印材料的大小，新建默认的图纸默认大小是：200x300mm。
