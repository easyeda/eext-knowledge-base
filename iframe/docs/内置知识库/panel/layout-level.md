# 层级

## 层级

面板中提供了若干不同的图层，例如打印层、透明控制层等。而每个图层可以绘制多个图元。层级切换是指在同一个图层中切换图元的显示顺序。

左侧对象树中，同一图层内，排列在上方的图元比下方的层级更高，而画布上会优先显示层级更高的图元。

![图 0](/storage/images/cn/panel/layout-level/layout-level_20240621_172123.png)

### 提升一层

**操作步骤**：

选择需要移动层级的图元

-   顶部菜单 - 布局 - 层级 - 提升一层
    ![图 1](/storage/images/cn/panel/layout-level/layout-level_20240621_172124.png)

或者

-   左侧面板 - 工程设计 - 对象，右键菜单 - 层级 - 提升一层
    ![图 2](/storage/images/cn/panel/layout-level/layout-level_20240621_172125.png)

或者

-   右键菜单 - 层级 - 提升一层
    ![图 14](/storage/images/cn/panel/layout-level/layout-level_20240621_172126.png)
