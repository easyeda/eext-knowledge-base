# 旋转

## 旋转

将图元以某个中心点为基准进行旋转。通过旋转，可以改变图元的方向或位置，使图元在面板画布上呈现不同的角度或朝向。

### 任意角度的左向旋转

**操作步骤**：

选中图元

-   顶部菜单 - 布局 - 旋转 - 旋转...
    ![图 0](/storage/images/cn/panel/layout-rotate/layout-rotate_20240621_172123.png)

或者

-   右键菜单 - 旋转 - 旋转...
    ![图 1](/storage/images/cn/panel/layout-rotate/layout-rotate_20240621_172124.png)

![图 6](/storage/images/cn/panel/layout-rotate/layout-rotate_20240621_172125.gif)

### 固定90°的左向旋转

**操作步骤**：

选中图元

-   顶部菜单 - 布局 - 旋转 - 左向旋转
-   全局快捷键：<kbd>Space</kbd>
    ![图 2](/storage/images/cn/panel/layout-rotate/layout-rotate_20240621_172126.png)

或者

-   顶部工具栏，直接点击左向旋转图标
    ![图 11](/storage/images/cn/panel/layout-rotate/layout-rotate_20240621_172127.png)

或者

-   右键菜单 - 旋转 - 左向旋转
    ![图 3](/storage/images/cn/panel/layout-rotate/layout-rotate_20240621_172128.png)

![图 10](/storage/images/cn/panel/layout-rotate/layout-rotate_20240621_172129.gif)
