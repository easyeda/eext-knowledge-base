# 翻转

## 翻转

将所选图元的方向或位置进行翻转，有左右翻转和上下翻转两种。

### 左右翻转

左右翻转就是在水平方向将图元左右颠倒，即原本在左边的部分以中心轴为对称移到右边，而原本在右边的部分以中心轴为对称移到左边。

**操作步骤**：

选中图元

-   顶部菜单 - 布局 - 翻转 - 左右翻转
-   全局快捷键：<kbd>X</kbd>
    ![图 0](/storage/images/cn/panel/layout-flip/layout-flip_20240621_172123.png)

**或者**

-   顶部工具栏，直接点击左右翻转图标
    ![图 1](/storage/images/cn/panel/layout-flip/layout-flip_20240621_172124.png)

**或者**

-   右键菜单 - 左右翻转
    ![图 2](/storage/images/cn/panel/layout-flip/layout-flip_20240621_172125.png)

![图 3](/storage/images/cn/panel/layout-flip/layout-flip_20240621_172126.gif)

### 上下翻转

上下翻转就是在垂直方向将图元上下颠倒，即原本在上方的部分以中心轴为对称移到下方，而原本在下方的部分以中心轴为对称移到上方。

**操作步骤**：

选中图元

-   顶部菜单 - 布局 - 翻转 - 上下翻转
-   全局快捷键：<kbd>Y</kbd>
    ![图 7](/storage/images/cn/panel/layout-flip/layout-flip_20240621_172127.png)

**或者**

-   顶部工具栏，直接点击上下翻转图标
    ![图 6](/storage/images/cn/panel/layout-flip/layout-flip_20240621_172128.png)

**或者**

-   右键菜单 - 上下翻转
    ![图 5](/storage/images/cn/panel/layout-flip/layout-flip_20240621_172129.png)
