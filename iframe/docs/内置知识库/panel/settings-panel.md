# 面板设置

## 面板设置

面板也可以像原理图和PCB一样支持一些通用和主题的设置。

### 操作步骤

-   顶部菜单 - 设置 - 面板/面板库，可以对面板/面板库设置一些个人偏好。

![图 0](/storage/images/cn/panel/settings-panel/settings-panel_20240621_172123.png)

可以设置画布网格类、网格尺寸、ALT吸附尺寸（在绘制过程中按ALT吸附应用该参数）等。

![图 1](/storage/images/cn/panel/settings-panel/settings-panel_20240621_172124.png)

也可以设置主题，点击设置字体颜色等。

![图 2](/storage/images/cn/panel/settings-panel/settings-panel_20240621_172125.png)

这些设置修改后应用时会应用到当前文件，需注意：下次新建文档的时候也会应用这些设置。
