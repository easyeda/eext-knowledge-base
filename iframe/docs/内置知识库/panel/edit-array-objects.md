# 阵列对象

## 阵列对象

阵列对象相当于批量复制对象，同时会将这些复制出来的对象按照设定好的行列数进行放置。

### 操作步骤

-   选择一个对象 - 顶部菜单 - 编辑 - 阵列对象...

![图 1](/storage/images/cn/panel/edit-array-objects/edit-array-objects_20240621_172123.png)

选择一个对象后，直接在顶部工具栏中单击阵列对象图标也可以进行阵列。

![图 6](/storage/images/cn/panel/edit-array-objects/edit-array-objects_20240621_172124.png)

在弹窗中输入需要阵列的行和列，还有行距和列距，单位默认为mm，不可修改。
填写完信息后点击确定，即可将所选对象按照填写的数值进行阵列排序。

![图 2](/storage/images/cn/panel/edit-array-objects/edit-array-objects_20240621_172125.png)

面板的阵列，不用受到当前网格尺寸的约束。即行距、列距支持输入任意的数值(精确到小数点后2位)，实际阵列时对象不会吸附格点，按照预设值进行排布。

![图 3](/storage/images/cn/panel/edit-array-objects/edit-array-objects_20240621_172126.png)

支持预览预设的阵列结果。

![图 4](/storage/images/cn/panel/edit-array-objects/edit-array-objects_20240621_172127.gif)

还可以选择通过鼠标调整确定阵列的行列间距。

![图 5](/storage/images/cn/panel/edit-array-objects/edit-array-objects_20240621_172128.gif)
