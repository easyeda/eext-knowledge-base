# 转换

## 转换

在多选图元后，右键可以转换图元形状类型。

![图 6](/storage/images/cn/panel/edit-convert/edit-convert_20240621_172123.png)

功能说明：

-   转换描边为填充形状：根据外轮廓生成一个填充图形。如图：矩形转为了两个图元，一个外轮廓无填充，一个有填充无外轮廓的矩形。
    ![图 7](/storage/images/en/panel/edit-convert/edit-convert_20240621_172123.png)

-   转换矩形和椭圆为路径：默认的矩形和椭圆是一个独立的图形，非路径组成，转为路径后成为形状。
    ![图 8](/storage/images/en/panel/edit-convert/edit-convert_20240621_172124.png)

-   形状合并：类似组合功能，把不同的图元或形状组合在一起。注意下图的选中效果和外轮廓线条。
    ![图 9](/storage/images/en/panel/edit-convert/edit-convert_20240621_172125.png)

-   形状打散：打散形状合并后的图元。
