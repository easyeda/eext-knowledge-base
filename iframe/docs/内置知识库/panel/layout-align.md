# 对齐

## 对齐

面板的对齐功能与原理图、复用图块、符号、封装、PCB的操作方式是一致的。

**对齐操作方法**：

-   选择需要对齐的的图元 - 顶部菜单 - 布局 - 对齐

![图 0](/storage/images/cn/panel/layout-align/layout-align_20240621_172123.png)

### 选择参考对象

支持自定义对齐功能是否选择参考对象。

勾选上"选择参考对象"菜单时，操作对齐都需要自己选择参考对象。

![图 18](/storage/images/cn/panel/layout-align/layout-align_20240621_172124.gif)

![图 15](/storage/images/cn/panel/layout-align/layout-align_20240621_172125.png)

![图 20](/storage/images/cn/panel/layout-align/layout-align_20240621_172126.png)

不勾选"选择参考对象"菜单时，操作对齐则是进行默认对齐的。

![图 19](/storage/images/cn/panel/layout-align/layout-align_20240621_172127.gif)

![图 16](/storage/images/cn/panel/layout-align/layout-align_20240621_172128.png)

### 左对齐

选择需要对齐的图元，需选择两个以上，单个没有参考点，是无法对齐的。

-   顶部菜单 - 布局 - 对齐 - 左对齐
-   快捷键：<kbd>ctrl</kbd>+<kbd>shift</kbd>+<kbd>L</kbd>

![图 1](/storage/images/cn/panel/layout-align/layout-align_20240621_172129.gif)

![图 2](/storage/images/cn/panel/layout-align/layout-align_20240621_172130.png)

### 对齐网格 {#对齐网格}

选择需要对齐的图元，将选中的图元对齐面板编辑器的网格。

-   顶部菜单 - 布局 - 对齐 - 对齐网格
-   快捷键：<kbd>ctrl</kbd>+<kbd>shift</kbd>+<kbd>G</kbd>
