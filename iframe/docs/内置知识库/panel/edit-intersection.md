# 交集

## 交集

我们可以通过交集功能对两个图元的相交部分进行截取。通过交集可以生成很多不好绘制的图形。简单来说就是：就是取A和B相交的地⽅。

**操作说明**：

-   选中元素 - 顶部菜单 - 编辑 - 交集
