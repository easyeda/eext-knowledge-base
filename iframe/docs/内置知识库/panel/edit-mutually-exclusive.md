# 互斥

## 互斥

通过互斥功能可以把将A和B相交部分去掉，保留不相交的部分。不同形状的图元的内部相交部分移除。

**功能入口**

-   顶部菜单 - 编辑 - 互斥。

![图 7](/storage/images/cn/panel/edit-mutually-exclusive/edit-mutually-exclusive_20240621_172123.png)

选中多个图元，点击互斥菜单即可。
