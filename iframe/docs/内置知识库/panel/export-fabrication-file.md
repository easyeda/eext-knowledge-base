# 导出面板制造文件

## 导出面板制造文件

嘉立创EDA专业版面板编辑器支持一键导出.epanm的面板制造文件。

**导出步骤**：

-   顶部菜单 - 文件 - 导出 - 面板制造文件；或顶部菜单 - 文件 - 导出 - 面板制造文件

![图 100](/storage/images/cn/panel/export-fabrication-file/export-fabrication-file_20240621_172123.png)

预览没有问题后，设置文件名，点导出按钮，会得到面板制造文件 epanm 文件。

![image-20220728154528808](/storage/images/cn/panel/export-fabrication-file/export-fabrication-file_20240621_172124.png)

导出制造文件前会进行文件规则检查，如果没有绘制必须的板框等图元，会在日志里面提示，根据提示绘制即可。

![图 107](/storage/images/cn/panel/export-fabrication-file/export-fabrication-file_20240621_172125.png)

导出面板制造文件后，前往立创商城下单：[面板打印下单](https://dos.szlcsc.com/dos/panel/print.html)
