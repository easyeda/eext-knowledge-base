# 仿真设置

嘉立创EDA专业版支持丰富的 NGspice 仿真参数配置项，包含瞬态、交流、直流扫描三类分析的设置。

## 功能入口

1. 顶部菜单栏 - 仿真 - 仿真设置

![图 0](/storage/images/cn/simulation/simulation-setting/simulation-setting_20251211_220025.png)

2. 顶部工具栏 - 仿真设置

## 参数说明
- 通用（NGspice）
  - 最大非线性：非线性器件迭代次数；默认适中；范围为正整数；建议在仿真收敛不佳时酌情提高，时间会增加。

- 瞬态分析（.tran）
  - 时间步长：每段时间的数据取值间隔；默认 `5u`；范围为正数时间；建议从 `5u` 起，过小会增加仿真时间。
  - 终止时间：仿真结束时间；默认 `25m`；范围为正数时间；建议根据波形周期设置为周期的若干倍。
  - 初始时间（选填）：采样起始时间；默认空；范围为非负时间；建议在忽略初始瞬态时使用。
  - 最大时间步长（选填）：允许的最大步长；默认空；范围为正数时间；建议用于控制数值稳定性。

- 交流分析（.ac）
  - 描述类型（Type）：频率扫描方式；可选 `Decade`、`Octave`、`Linear`；默认 `Decade`；建议一般使用 `Decade`。
  - 每十倍频程点数（Decade）：当选择 `Decade` 时显示；默认 `10`；范围为正整数；建议 `10–20` 平衡精度与速度。
  - 每八倍频程点数（Octave）：当选择 `Octave` 时显示；默认 `10`；范围为正整数；建议 `10–20`。
  - 点数（Linear）：当选择 `Linear` 时显示；默认 `10`；范围为正整数；建议根据频点分布选择。
  - 起始频率：扫描起点；默认 `500`；范围为非负频率；建议结合电路工作频率设定。
  - 停止频率：扫描终点；默认 `10G`；范围 `≤ 10G`；建议不超过项目允许的最高频率。

- 直流扫描（.dc）
  - 扫描源（源1/源2）：从电路中的电压源和电流源中选择；名称为器件位号；默认必须选择源1；范围为当前图页内的源列表；建议源1与源2不重复，源1选择后源2中隐藏同名项。
  - 起始值：扫描起点；默认空；范围为数值（单位随器件类型）；建议覆盖关注的工作点范围。
  - 停止值：扫描终点；默认空；范围为数值；建议略大于工作点上限。
  - 增量（步长）：取样间隔；默认空；范围为正数；建议与需要的曲线分辨率匹配；“增量与步长一致”。
