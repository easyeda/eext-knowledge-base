# 命名及单位说明

本章节阐述仿真过程中采用的命名规范：

-   Spice模型文本默认不区分大小写；

-   字段分隔符包含一个或多个空格、逗号、等号（=）、左括号或右括号，冗余空格将被忽略；

-   续行符采用首列"+"（加号）标识；

-   有效字符集限定为：a-z、0-9及下划线"_"；

-   标识符必须以字母（a-z）起始且不得包含分隔符。数值字段支持整数（12，-44）、浮点数（3.14159）及科学计数法表示。

## 常用单位说明

| 后缀 |   名称   | 指数因子 |     | 后缀 | 名称 |   指数因子   |     | 后缀 | 名称 | 指数因子 |
| :--: | :------: | :------: | :-: | :--: | :--: | :----------: | :-: | :--: | :--: | :------: |
|  T   | 太（拉） |  10^12   |     |  K   |  千  |     10^3     |     |  u   |  微  | 10^(-6)  |
|  G   |   千兆   |   10^9   |     | mil  | 密尔 | 25.4×10^(-6) |     |  n   |  纳  | 10^(-9)  |
| MEG  |    兆    |   10^6   |     |  m   |  毫  |   10^(-3)    |     |  p   |  皮  | 10^(-12) |

**注意**：尾随数字的非比例因子字母及紧随比例因子的字母均被忽略。

1000、1000.0、1000Hz、1e3、1.0e3、1kHz、1k表示相同数值。

**注意**：M/m表示毫（1e-3），兆必须使用MEG表示（1e6）。

参考节点必须定义为0，GND节点默认作为接地节点，系统内部将其视为全局节点并映射至0平面。
