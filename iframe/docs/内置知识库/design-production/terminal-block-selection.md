# 接线端子选型

## 插拔式接线端子

- 插拔式接线端子（Plug-in Terminal Block）常用于电源板、控制板、传感器、驱动器等场合。
- 它的核心特点就是“可插拔”；2P、3P、...到20P任意组合；
- 具有直插式、斜插式、横插式等多种排布；
- 耐电流能力强，从尺寸2.54mm、3.5mm、3.81mm到5.08mm、7.62mm等；间距越大往往支持的电流越大；例如3.81mm额定电流在7A以上；
- 接口插头插座之间有机械定位结构，防松脱，且有绝缘壳体避免短路。

## 螺钉式接线端子

- 接线牢固，直接通过螺钉压紧导线或压片，机械强度高；
- 常见间距有5.08mm、7.68mm、9.52mm、10.16mm等，往往间距越大，能承受的电压和电流越高，以5.08mm-2P接口为例，额定电压通常在250V、15A以上；
- 兼容多种线材，只要能拧进去就能固定；

## 弹簧式接线端子

- 无螺丝结构，接线速度快，把线推进去即可自动夹紧
- 外观紧凑，多采用小间距设计，如2.5mm、3.5mm、5.08mm；
- 抗震动效果好，弹簧结构长期稳定；

## 栅栏式接线端子

- 采用绝缘隔板设计，安全性高；
- 通常用于高压250V以上、大电流10A以上的环境，在工业电源、逆变器、大功率设备中常用；
- 通过螺钉式压片固定，拉力大，通常会采用大螺钉接口，接触面大
- 大部分型号会带透明防护罩，防触电、电弧散射；
- 间距大、通常在7.62mm、9.5mm以上；
