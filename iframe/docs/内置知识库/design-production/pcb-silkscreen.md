# PCB常见丝印标识

## 1 元件位号标识

- ANT：天线（Antenna）；
- B ：电池（Battery）；
- BT：蓝牙模块（Bluetooth）；
- BZ：蜂鸣器（Buzzer）；
- C : 电容（Capacitor）；
- D : 二极管（Diode）；
- DS：显示器件（Display）;
- F : 保险丝（Fuse）；
- FB：磁珠滤波器（Ferrite Bead）；
- FS：快速熔断保险丝（Fast Blow Fuse）;
- H : 排针排母（Header）；
- IR：红外二极管（Infared Diode）；
- JP：跳线（Jumper）；
- K : 继电器（Relay）；
- KEY：按键（Key）；
- L ：电感（Inductance）；
- M ：电机（Motor）；
- MIC：麦克风（Microphone）；
- NTC: 负温度系数热敏电阻（Negative Temperature Coefficient Thermistor）;
- LDR：光敏电阻（Light Dependent Resistor）；
- LED：发光二极管（Light Emitting Diode）；
- Q ：晶体管/三极管/场效应管（Transistor）；
- R : 电阻（Resistor）；
- RP：电位器/可调电阻；
- RN：排阻（Resistor Network）；
- RT：热敏电阻（Thermistor）；
- S ：开关（Switch）；
- T ：变压器（Transformer）；
- TP：测试点（Test Point）；
- U ：集成电路（IC）；
- USB：USB接口；
- X : 晶振（Crystal Oscillator）；
- ZD：稳压二极管/齐纳二极管（Zener Diode）

## 2 常见认证标识

- CC知识产权（Creative Commons）

<center>
<img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-silkscreen/pcb-silkscreen_20250321_195802.png"width="150">
</center>

&emsp;&emsp;CC知识产权是一种灵活的版权许可体系，旨在帮助创作者在保留部分权利的同时，允许他人合法共享和使用其作品。与传统版权“保留所有权利”不同，CC许可提供“保留部分权利”的选择，使作品能在特定条件下被广泛传播和使用。

- CE认证（Conformité Européenne）

<center>
<img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-silkscreen/pcb-silkscreen_20250324_113609.png"width="150">
</center>

&emsp;&emsp;CE认证是欧盟对产品符合其安全、健康、环保等要求的强制性认证标志。产品贴有CE标志，表明其符合欧盟相关法规，可在欧洲经济区（EEA）内自由流通。

- 3C认证（China Compulsory Certification）

<center>
<img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-silkscreen/pcb-silkscreen_20250324_112833.png"width="150">
</center>

&emsp;&emsp;3C认证的全称为“中国强制性产品认证”，是中国政府为保护消费者人身安全和国家安全、加强产品质量管理、依照法律法规实施的一种产品合格评定制度。

- UL认证（Underwrites Laboratories）

<center>
<img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-silkscreen/pcb-silkscreen_20250324_115243.png"width="150">
</center>

&emsp;&emsp;UL标志，是美国保险商实验室对机电包括民用电器类产品颁发的安全保证标志，不论是从美国出口或进入美国市场的产品都必须有该标志。

- FCC认证（Federal Communications Commission）

<center>
<img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-silkscreen/pcb-silkscreen_20250324_112258.png"width="150">
</center>

&emsp;&emsp;FCC认证，是美国联邦通讯委员会对进入美国市场的电子电器产品在EMC、Telecom等方面作了明确要求，电子电器产品需拿到FCC认证证书才能顺利进入美国市场。

- SGS认证（ Societe Generale de Surveillance S.A. ）

<center>
<img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-silkscreen/pcb-silkscreen_20250324_113058.png"width="150">
</center>

&emsp;&emsp;SGS认证指的是通用公证行，SGS根据标准、法规、客户要求等条件对目标进行符合性认证的服务。

- TÜV认证（Technischer Überwachungs-Verein）

<center>
<img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-silkscreen/pcb-silkscreen_20250324_113320.png"width="150">
</center>

&emsp;&emsp;TÜV标志是德国技术检验协会，专为元器件产品定制的一个安全认证标志，在德国和欧洲得到广泛的接受。

- ISO认证（International Organization for Standardization）

<center>
<img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-silkscreen/pcb-silkscreen_20250324_114740.png"width="150">
</center>

&emsp;&emsp;ISO认证，全称为“国际标准化组织认证”，是全球范围内通用的认证体系。

- RoHS认证（Restriction of Hazardous Substances）

<center>
<img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pcb-silkscreen/pcb-silkscreen_20250324_152215.png"width="150">
</center>

&emsp;&emsp;ROHS认证一般指RoHS。 RoHS是由欧盟立法制定的一项强制性标准，它的全称是《关于限制在电子电气设备中使用某些有害成分的指令》(Restriction of Hazardous Substances)。RoHS检测项目有 铅(Pb)，镉(Cd)，汞(Hg)，六价铬(Cr6+)，多溴联苯(PBBs)和多溴二苯醚(PBDEs)。
