# 六、AOI

&emsp;&emsp;AOI（Automated Optical Inspection）指自动光学检测。AOI的检测原理是将待检测的电路板放到AOI仪器检测面板上，AOI仪器开始自动扫描电路板，并与MI文件中的相应文件进行对比，一旦有区别，就会停下来，把图像放大显示到显示器上，如图6-1所示。

&emsp;&emsp;AOI检测只是针对电路板的外观进行检测，可以很好的检测出开路、短路和断路的地方。不过，过孔是否全部导通，这里就无法检测了，过孔检测需要在后面的“飞针测试”或“复合模具测试”中完成。
