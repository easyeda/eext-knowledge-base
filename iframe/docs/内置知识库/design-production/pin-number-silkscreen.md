# PCB位号丝印快速调整

‌&emsp;&emsp;丝印是PCB表面的文字说明标记，模糊混乱及残缺的丝印可能会造成很多严重后果，例如元器件焊接错位置，极性器件焊反方向，不方便调试维修等。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_202353.png"  style="border:2px solid gray" width="500"></center>

‌&emsp;&emsp;在PCB设计优化的最后阶段，需要对元件位号丝印进行调整，在调整元件位号丝印时，需要注意以下原则：

‌&emsp;&emsp;①丝印需要清晰，字体调整到合适尺寸，推荐字符高度需要≥1mm，字符粗细≥0.15mm，以确保印制后可读性。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_155728.png"  style="border:2px solid gray" width="500"></center>

‌&emsp;&emsp;可选中一个位号丝印后，右键选择查找，种类可下拉选择到元件属性-位号，属性按需选择配置，然后点击查找全部，在右侧属性面板里，可设置元件属性的字体，线宽和高等，快速批量调整器件的位号丝印。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_150629.png"  style="border:2px solid gray" width="500"></center>

‌&emsp;&emsp;默认字体线宽为6mil，高为45mil，除默认外的其他字体线宽暂不支持设置，线宽与高是绑定的，修改高后线宽也会同步变换。推荐字符高度需要≥1mm，字符粗细≥0.15mm，以确保印制后可读性。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_173327.png"  style="border:2px solid gray" width="300"></center>

‌&emsp;&emsp;推荐使用非衬性字体，非衬性字体会更加圆润饱满些，线条简洁，笔画末端无装饰线。衬性字体笔画末端有装饰线，局部线条过于纤细影响辨别，印制出来的丝印字符易出现锯齿等问题。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_162320.png"  style="border:2px solid gray" width="800"></center>

‌&emsp;&emsp;②丝印方向需要统一，不超过两个朝向摆放，避免焊接或维修时需频繁旋转查看。 ‌

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_202039.png"  style="border:2px solid gray" width="500"></center>

‌&emsp;&emsp;可以在点击布局-属性位置，快速批量调整器件的属性位置。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_200721.png"  style="border:2px solid gray" width="200"></center>

‌&emsp;&emsp;在属性位置弹窗里面，可以选择仅选中元件或全部元件，下拉属性/属性值选型，对元件位号等快速调整位置，属性位置提供上边、下边、右边、左边、中间、左上、左下、右上、右下几种方向。
<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_200818.png"  style="border:2px solid gray" width="800"></center>

‌&emsp;&emsp;元件的位号丝印快速放置到元件的左边。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_215738.png"  style="border:2px solid gray" width="300"></center>

‌&emsp;&emsp;③丝印不要放置焊盘上或开阻焊的区域，避免丝印缺失。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_204523.png"  style="border:2px solid gray" width="800"></center>

‌&emsp;&emsp;在PCB空白位置按照元件摆放放置位号丝印，或使用辅助线等进行标记，方便阅读。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_213706.png"  style="border:2px solid gray" width="800"></center>

‌&emsp;&emsp;④丝印处尽量不要打过孔，避免丝印残缺，不方便辨认。

<center><img src="/storage/images/zh-hans/design-production/pcb-design/knowledge-point/pin-number-silkscreen/pin-number-silkscreen_20251125_204209.png"  style="border:2px solid gray" width="500"></center>
