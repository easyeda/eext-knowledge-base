# 什么是增益

&emsp;&emsp;“增益”表示“输出信号与输入信号的比值”，这一概念比放大倍数更加严谨，“增益”既可以表示放大（增益>=1），也可以表示衰减（增益<1），能够完美的表示电子电路中器件（如运算放大器）的工作状态。

# 为什么增益会普及

&emsp;&emsp;在运算放大器中，开环增益极高（通常为10⁵~10⁷），此时如果使用倍数，会非常大，而使用对数单位“分贝dB”表示则显得更为简洁。
<br/>
几种常用的增益的分贝定义为：

- 电压增益：GdB = 20lg（Vout/Vin）；
- 功率增益：GdB = 10lg（Pout/Pin）；
- 声压增益：Gdb = 20lg（声压Pa/参考声压20ɥPa）；

# 为什么不同公式中增益系数不一样

&emsp;&emsp;电压增益用20系数、功率增益用10系数的主要原因是物理量的不同；分贝最初是为了描述功率的相对变化，前面的10系数是约定俗成的，被广泛遵守；

- 分贝原始定义为：GdB = 10lg（Pout/Pin）；
- 通用公式为：XdB = 10lg（实际值/参考值）；

&emsp;&emsp;电压本身不等于功率，但是P=（V²）/R；此时换算为Vout/Vin时，就可以把2提出来放置前面，就变成了20系数；它是从功率增益的定义推导而来的；因为电压与功率存在平方关系，同理电流也是20系数。
<br/>
&emsp;&emsp;其他公式的系数可能会有所区别，但大部分都与能量有关，比如声压增益，这时声压属于“场量”，其平方与能量成正比。所以也是20；核心同样是输出与输入的比值。

# 运算放大器中的特殊增益点

- 增益交叉点：指运算放大器的开环增益幅值下降到1（即0dB）时的频率点，该点在分析运算放大器的稳定性时非常重要，当系统的环路增益在该频率点处相移超过180°时，闭环放大电路可能会产生振荡。
- 单位增益频率点：是指开环增益下降到0dB时对应的频率，也称为单位增益带宽，与增益带宽积有关，通常增益带宽积等于单位增益频率\*直流开环增益，在该频率点内，运算放大器可保持一定的放大能力。
- 滤波器中的截止频率点：定义了滤波器“有效工作范围”与“衰减范围”的分界，表示信号在该频率点的功率衰减为输入功率的一半，电压衰减为输入电压的0.707倍；
    - 对于低通滤波器而言，-3dB是高频开始显著衰减的起点，低于该频率的信号衰减小于3dB能够有效通过；
    - 对于高通滤波器而言，-3dB是低频开始显著衰减的起点，高于该频率的信号衰减小于3dB能够有效通过；
    - 对于带通/带阻滤波器而言：-3dB定义了通带/阻带的边界，通带内信号衰减小于3dB，阻带内信号衰减大于3dB；

# 常用增益换算为倍数

&emsp;&emsp;下述换算增益系数均为20

- 0dB：放大倍数为1
- 3dB：放大倍数为1.4
- 6dB：放大倍数为2
- 9dB：放大倍数为2.8
- 12dB：放大倍数为4
- 18dB：放大倍数为8
- 20dB：放大倍数为10
- -3dB：放大倍数为0.707倍
- -6dB：放大倍数为0.5倍
- -10dB：放大倍数为0.1倍
- -20dB：放大倍数为0.01倍
- -60dB：放大倍数为0.001倍
