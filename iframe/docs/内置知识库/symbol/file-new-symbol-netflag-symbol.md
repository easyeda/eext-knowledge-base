# 网络标识符号

如果系统提供的几个网络标识符不满足你的需要，你可以创建自己常用的网络标识符号样式，后续方便复用。

入口：顶部菜单 - 新建 - 符号，类型选择网络标识符号。

![图 0](/storage/images/cn/symbol/file-new-symbol-netflag-symbol/file-new-symbol-netflag-symbol_20250213_092320.jpg)

绘制样式后，放置一个引脚。不能放置多过已引脚。

保存后可以在底部面板的元件库找到：底部面板 - 元件库 - 符号 - 个人。

**注意**：必须创建一个同名或者不同名的器件绑定创建的好的符号，才可以在原理图里面放置器件进行使用。
