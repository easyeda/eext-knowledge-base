# 引脚属性

## 引脚属性

原理图符号的引脚是元件最重要的组成部分之一，它允许进行导线连接以构成整个电路。

**顶部工具栏 - 放置 - 引脚 或 使用快捷键 <kbd>Ait</kbd> + <kbd>P</kbd>**

## 引脚属性

当你选中一个引脚时，可以在右边的属性面板查看和修改它的属性。

![图 1](/storage/images/cn/symbol/side-panel-right-pin-property/side-panel-right-pin-property_20240621_172124.png)

-   **名称**：引脚的名字，如上图选中的引脚名字为“VCC”。
-   **引脚编号**：如上图选中的引脚编号为“8”。**该编号与元件对应的封装的焊盘编号相对应，并且两者的编号需要一致。**你还可以使用带字母的编号，如 A1, B1, C1, A2, B2等。还可以用逗号分隔设置多个引脚编号，所设置的引脚与多个焊盘对应。
-   **ID**：引脚的ID是系统固定的，不可更改。
-   **引脚类型**：可选择引脚类型：输出、双向、无源、开集电集、开发射集、电源、地、高阻、信号终端、无定义

-   **引脚形状**：可选择引脚的附加形状：无、反相、时钟、反相时钟。

![图 2](/storage/images/cn/symbol/side-panel-right-pin-property/side-panel-right-pin-property_20240621_172125.png)

-   **非链接标识**：设置为是则会在引脚上显示非连接标识。

![图 3](/storage/images/cn/symbol/side-panel-right-pin-property/side-panel-right-pin-property_20240621_172126.png)

-   **引脚方向**：只支持上下左右方向选择，其他方向暂不支持。

-   **长度**：引脚长度。

-   **引脚颜色**：可以修改引脚的颜色。

-   **新增属性**：可以在更多属性底部下拉框添加自定义属性。
