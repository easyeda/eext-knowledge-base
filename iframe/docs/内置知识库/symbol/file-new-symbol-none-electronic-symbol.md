# 无电气符号

如果你想创建logo库或其他标记库，后续方便复用，可以创建无电气符号。

入口：顶部菜单 - 新建 - 符号，类型选择无电气符号。

![图 61](/storage/images/en/symbol/file-new-symbol-none-electronic-symbol/file-new-symbol-none-electronic-symbol_20240621_172123.png)

无电气符号不能放置引脚。

保存后可以在底部面板的元件库找到：底部面板 - 元件库 - 符号 - 个人。

**注意**：必须创建一个同名或者不同名的器件绑定创建的好的符号，才可以在原理图里面放置使用。
