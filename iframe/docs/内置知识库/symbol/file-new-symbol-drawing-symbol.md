# 图纸符号

如果系统提供的几个图纸符号不满足你的需要，你可以创建自己常用的图纸符号样式，后续方便复用。

入口：
1）在开始页，新建器件，选择新建图纸类型
![图 1](/storage/images/cn/symbol/file-new-symbol-drawing-symbol/file-new-symbol-drawing-symbol_20240621_172124.png)

2）顶部菜单 - 新建 - 图纸，类型选择图纸符号。
![图 2](/storage/images/cn/symbol/file-new-symbol-drawing-symbol/file-new-symbol-drawing-symbol_20240621_172125.png)

注意：图纸符号不能放置引脚。

创建步骤：

1、新建图纸符号后，在左侧属性设置图纸所需要的属性，勾选需要显示在画布的值。带有 @ 开头的属性是系统内置的属性，在放置在图页后，这些属性会自动更新，不需要预先设置值。

![图 66](/storage/images/en/symbol/file-new-symbol-drawing-symbol/file-new-symbol-drawing-symbol_20240621_172124.png)

2、可以手动调整图纸边界，右下角表头等的线条大小和位置，也可以使用左侧的向导进行创建

![图 67](/storage/images/en/symbol/file-new-symbol-drawing-symbol/file-new-symbol-drawing-symbol_20240621_172125.png)

3、调整完成后，保存。保存后可以在底部面板的元件库找到：底部面板 - 元件库 - 符号 - 个人。

**注意**：必须创建一个同名或者不同名的器件绑定创建的好的符号，才可以在原理图里面放置器件进行使用。
