# 创建符号

## 创建符号库

创建步骤

-   顶部菜单 - 文件 - 新建 - 符号

![image-20210521111044245](/storage/images/en/symbol/file-new-symbol/file-new-symbol_20240621_172123.png)

-   **归属**：选择符号的使用者或创建者。

-   **标题**：创建符号的名称。

-   **类型**：可从创建其他类型的符号，例如：元件符号、网络标识符号、网络端口符号、图纸符号、无电气符号。具体符号类型的创建请看下一章节。

-   **分类**：选择好分类更容易区分器件，更好的管理

-   **描述**：对符号的一些详细描述。

点击确定即可创建符号

![image-20210521111439687](/storage/images/en/symbol/file-new-symbol/file-new-symbol_20240621_172124.png)

所有创建的符号必须被器件关联才可以放置在原理图中。所以创建符号后需要创建器件进行关联。
