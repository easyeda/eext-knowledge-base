# 符号向导

嘉立创EDA专业版符号向导器可以快速创建符号，目前只支持IC类型的DIP、QFP和SIP三种类型的符号快速创建。

操作步骤：

1、在符号编辑器界面左侧的视图，选择向导

![image-20210521112122620](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172123.png)

2、选择需要创建符号的类型

![image-20210521112224971](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172124.png)

**3、设置原点**

原点在第一脚：将符号的第一引脚设为原点，第一脚的坐标值都为0。

![image-20201217100147777](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172125.png)

原点设置在中心：将符号的中心位置设置成原点，符号的中心位置坐标值为0。

![image-20201217100147777](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172125.png)

填写好相应的引脚数量

![image-20201217100605857](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172126.png)

设置引脚的间距

![image-20201217100840119](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172127.png)

**注意**：引脚间距不可设置太小，否则生成符号时，会导致像下方图片一样，引脚名重叠，**推荐间距0.15inch。**

![image-20201217100840119](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172127.png)

引脚编号的方向

![image-20201217100840119](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172127.png)

编号方向为逆时针圆生成

![image-20201217100840119](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172127.png)

引脚编号方向从做到右，从上到下生成的。两个的区别在于引脚的生成序号位置不同，更适用于其他类型引脚编号方向的符号。

![image-20201217101425401](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172128.png)

点击生成

![image-20201217101425401](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172128.png)

选择是则清空你当前符号编辑器里的的符号，并且生成。

选择否则不生成符号。

![image-20201217101622427](/storage/images/en/symbol/side-panel-left-wizard/side-panel-left-wizard_20240621_172129.png)

生成的DIP类型符号
